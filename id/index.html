FAQ

Pertanyaan yang Sering Diajukan

Bagaimana aplikasi analitik lari ini mendapatkan data saya?

RunAnalytics sinkronisasi dengan Apple Health untuk mengimpor latihan lari yang direkam oleh perangkat atau aplikasi yang kompatibel. Ini termasuk jam tangan pintar, pelacak kebugaran, dan entri manual. Aplikasi memproses data ini secara lokal untuk menghitung metrik performa lari tingkat lanjut.

Apa itu tes kecepatan lari kritis dan bagaimana cara melakukannya?

Critical running speed (CRS) adalah protokol ilmiah menggunakan 2 lari upaya maksimum: 5K dan 3K dengan istirahat 10-20 menit di antara keduanya. Aplikasi menghitung ambang aerobik Anda dari waktu ini dan secara otomatis menyesuaikan semua zona latihan lari. Ulangi setiap 6-8 minggu untuk melacak progres.

Apakah data lari saya diunggah ke cloud?

Tidak. RunAnalytics memproses semua data lari secara lokal di iPhone Anda. Tidak ada server eksternal, tidak ada akun cloud, tidak ada transfer data. Anda mengontrol ekspor: hasilkan file JSON, CSV, HTML, atau PDF dengan metrik performa lari Anda dan bagikan sesuka Anda.

Bisakah saya menggunakan aplikasi analitik lari ini untuk lari trail?

Ya. RunAnalytics bekerja dengan latihan lari apa pun di Apple Health, termasuk lari trail. Aplikasi menyesuaikan metrik efisiensi lari dengan data yang tersedia baik Anda di trek atau trail, memberikan analisis performa yang relevan untuk setiap lingkungan.

Apa perbedaan antara paket bulanan dan tahunan?

Kedua paket menawarkan fitur yang identik: semua metrik performa lari, zona latihan tanpa batas, perbandingan temporal, ekspor ganda, dan pembaruan gratis. Satu-satunya perbedaan adalah harga: tahunan menghemat 18% (setara dengan ‚Ç¨3.25/bulan vs ‚Ç¨3.99/bulan).
