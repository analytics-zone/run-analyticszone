Vetenskapliga löpvärden

Kritisk löphastighet (CRS) avgör din aeroba tröskel, vilket möjliggör beräkning av Training Stress Score (TSS) och prestationsmätning CTL/ATL/TSB baserat på beprövad idrottsforskning.

Prestationsjämförelser

Jämförelser över tid på vecko-, månads- och årsbasis med automatisk trendanalys och procentuella förändringar för alla prestationsmått.

Fullständigt integritetsskydd

All löpdata behandlas lokalt på din iOS-enhet. Inga servrar, ingen molnlagring, ingen spårning. Du har full kontroll över din löpanalys.

Exportera vart som helst

FAQ

Vanliga frågor

Hur hämtar appen mina löpdata?

Run Analytics synkroniserar med Apple Hälsa för att importera löppass registrerade med valfri kompatibel enhet eller app, inklusive smartklockor, träningsarmband och manuella inmatningar. Appen behandlar dessa data lokalt för att beräkna avancerade prestationsmått.

Vad är testet för kritisk löphastighet och hur gör jag det?

Kritisk löphastighet (CRS) är ett vetenskapligt protokoll som använder två maxlöpningar: 5 km och 3 km med 10-20 minuters vila emellan. Appen beräknar din aeroba tröskel från dessa tider och ställer in alla träningszoner automatiskt. Repetera var 6-8:e vecka.

Laddas mina löpdata upp till molnet?

Nej. Run Analytics behandlar alla löpdata lokalt på din iPhone. Inga externa servrar, inga molnkonton, inga dataöverföringar. Du styr exporten: skapa JSON-, CSV-, HTML- eller PDF-filer med dina mätvärden och dela dem som du vill.

Kan jag använda denna analysapp för terränglöpning?

Ja. Run Analytics fungerar med alla löppass i Apple Hälsa, inklusive terränglöpning. Appen anpassar effektivitetsmåtten till tillgängliga data, oavsett om det är på bana eller i skogen, och ger relevanta prestationsanalyser för alla miljöer.

Vad är skillnaden mellan månads- och årsplanen?

Båda planerna erbjuder identiska funktioner: alla mätvärden, obegränsade zoner, tidsjämförelser, flera exportformat och gratis uppdateringar. Den enda skillnaden är priset: årsplanen sparar dig ca 17 % (motsvarar ca 41 kr/månad jämfört med 49 kr/månad).

Kan jag avsluta min prenumeration när som helst?

Ja. Prenumerationer hanteras via App Store, så du kan avsluta när som helst under Inställningar → [Ditt namn] → Prenumerationer. Om du avslutar behåller du åtkomsten fram till slutet av din nuvarande faktureringsperiod.

Redo att ta din löpprestanda till nästa nivå?

Gör som tusentals andra löpare som använder vetenskaplig analys för att förbättra sina resultat. Börja din 7 dagars gratis provperiod med vår integritetsfokuserade iOS-app idag.
