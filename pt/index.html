FAQ Section
FAQ

Perguntas Frequentes

Como é que este app de análise de corrida obtém os meus dados?

O Run Analytics sincronizado com o Apple Health para importar treinos de corrida gravados por qualquer dispositivo ou aplicativo compatível. Isto inclui smartwatches, monitores de atividade e entradas manuais. Um aplicativo processa estes dados localmente pará calcular métricas de desempenho avançadas.

O que é o teste de velocidade crítica e como o faço?

A velocidade crítica de corrida (CRS) é um protocolo científico que utiliza 2 corridas de esforço máximo: 5K e 3K com 10-20 minutos de descanso entre elas. Um aplicativo calcula seus limites aeróbicos a partir destes tempos e ajustes automaticamente todas como zonas de treino. Repita a cada 6-8 semanas para acompanhar o progresso.

Os meus dados de corrida são carregados para a nuvem?

Não. O Run Analytics processa todos os dados localmente no seu iPhone. Não existem servidores externos, nem contas na nuvem, nem transferências de dados. Você controla as exportações: gere arquivos JSON, CSV, HTML ou PDF com como suas considerações e partes como entender.

Posso usar este aplicativo de análise para corrida em trilha?

Sim. O Run Analytics funciona com qualquer treino de corrida no Apple Health, incluindo corrida em trilha. Um aplicativo adaptar as estatísticas de eficiência aos dados disponíveis, seja em pista ou em trilho, fornecidas análises de desempenho relevantes para cada ambiente.

Qual é a diferença entre os planos mensais e anuais?

Ambos os planos oferecem funcionalidades idênticas: todas as métricas de desempenho, zonas de treino ilimitadas, comparações temporais, múltiplas exportações e atualizações gratuitas. A única diferença é o preço: o anual permite poupar 18% (equivalente a 3,25€/mês vs 3,99€/mês).

Posso cancelar a assinatura a qualquer momento?

Sim. As assinaturas são gerenciadas através da App Store, pelo que você pode cancelar quando quiser em Definições → [Seu nome] → Assinaturas. Se cancelar, manterá o acesso até o final do período de faturação atual.
