🏃ပရော်ဖက်ရှင်နယ် ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှု

ပိုမိုဉာဏ်ကောင်းစွာ လေ့ကျင့်ပါ၊ ပိုမြန်စွာ ပြေးပါ

ကိုယ်ရေးကိုယ်တာ ဦးစားပေး iOS အက်ပ်တွင် သိပ္ပံနည်းကျ ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်များ၊ ပုဂ္ဂိုလ်ရေး လေ့ကျင့်ရေး ဇုန်များနှင့် ပြည့်စုံသော စွမ်းဆောင်ရည် ခြေရာခံခြင်း။ သင့် iPhone တွင် ဒေတာအားလုံး ပြည်တွင်းတွင် လုပ်ဆောင်ပြီး ကိုယ်ရေးကိုယ်တာ အပြည့်အဝ ကာကွယ်ထားသည်။

ကျွန်ုပ်တို့၏ ဖြေရှင်းနည်းများကို လေ့လာပါ

✓ ၇ ရက် အခမဲ့ စမ်းသုံးခွင့်    ✓ အကောင့် မလိုအပ်ပါ    ✓ ဒေတာ ၁၀၀% ပြည်တွင်း

အင်္ဂါရပ်များ

အဆင့်မြင့် ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်များ

အဆင့်တိုင်းရှိ ပြေးသမားများအတွက် ဒီဇိုင်းထုတ်ထားသော ပရော်ဖက်ရှင်နယ် ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှု

သိပ္ပံနည်းကျ ပြေးခြင်း မက်ထရစ်များ

Critical Running Speed (CRS) သည် သင့် aerobic threshold ကို ဆုံးဖြတ်ပေးပြီး Training Stress Score (TSS) တွက်ချက်မှုနှင့် CTL/ATL/TSB စွမ်းဆောင်ရည် ခြေရာခံခြင်း ကို အားကစား သိပ္ပံ သုတေသန အပေါ် အခြေခံ၍ ဖွင့်ပေးသည်။

ပုဂ္ဂိုလ်ရေး လေ့ကျင့်ရေး ဇုန်များ

သင့် critical running speed အပေါ် ချိန်ညှိထားသော ပြေးခြင်း လေ့ကျင့်ရေး ဇုန် ၇ ခု။ နာလန်ထူခြင်း၊ aerobic ဖွံ့ဖြိုးမှု၊ threshold လေ့ကျင့်ခြင်း သို့မဟုတ် VO₂max တိုးတက်မှုအတွက် လေ့ကျင့်မှုတိုင်းကို အကောင်းဆုံးဖြစ်အောင် လုပ်ပါ။

စွမ်းဆောင်ရည် နှိုင်းယှဉ်မှုများ

ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်အားလုံးအတွက် အလိုအလျောက် လမ်းကြောင်း ရှာဖွေမှုနှင့် ရာခိုင်နှုန်း ပြောင်းလဲမှုများဖြင့် အပတ်စဉ်၊ လစဉ်နှင့် နှစ်စဉ် ကာလ နှိုင်းယှဉ်မှုများ။

ကိုယ်ရေးကိုယ်တာ အပြည့်အဝ ကာကွယ်မှု

ပြေးခြင်း ဒေတာအားလုံး သင့် iOS စက်တွင် ပြည်တွင်း၌ လုပ်ဆောင်သည်။ ဆာဗာမရှိ၊ cloud သိုလှောင်မှုမရှိ၊ ခြေရာခံမှုမရှိ။ သင့် ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှုကို အပြည့်အဝ ပိုင်ဆိုင်ပြီး ထိန်းချုပ်နိုင်သည်။

မည်သည့်နေရာသို့မဆို တင်ပို့ပါ

JSON၊ CSV၊ HTML သို့မဟုတ် PDF ဖော်မတ်များဖြင့် လေ့ကျင့်မှုများနှင့် ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်များကို တင်ပို့ပါ။ နည်းပြများ၊ spreadsheets နှင့် လေ့ကျင့်ရေး ပလက်ဖောင်းများနှင့် တွဲဖက်အသုံးပြုနိုင်သည်။

ချက်ချင်း စွမ်းဆောင်ရည်

ပြည်တွင်း-ဦးစားပေး ဗိသုကာဖြင့် အက်ပ် စတင်ချိန် ၀.၃၅ စက္ကန့်အောက်။ sync သို့မဟုတ် ဒေါင်းလုဒ်များကို စောင့်စရာမလိုဘဲ သင့် ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှုကို ချက်ချင်း ကြည့်ရှုပါ။

Run Analytics ဘာကြောင့်

အလေးအနက် အားကစားသမားများအတွက် ကိုယ်ရေးကိုယ်တာ-ဦးစားပေး ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှု

ရှုပ်ထွေးမှုမရှိဘဲ ပရော်ဖက်ရှင်နယ် ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်များ

🎯

CRS စမ်းသပ်မှု ပရိုတိုကော

သင့် critical running speed ကို ဆုံးဖြတ်ရန် ပေါင်းစပ်ထားသော 1200m/3600m စမ်းသပ်မှု ပရိုတိုကော။ တိုးတက်မှုကို ခြေရာခံရန်နှင့် ပြေးခြင်း လေ့ကျင့်ရေး ဇုန်များကို အလိုအလျောက် ချိန်ညှိရန် ၆-၈ ပတ်တိုင်း ထပ်လုပ်ပါ။

📱

Native iOS ပြေးခြင်း အက်ပ်

ချောမွေ့သော စွမ်းဆောင်ရည်နှင့် iOS ပေါင်းစပ်မှုအတွက် SwiftUI ဖြင့် တည်ဆောက်ထားသည်။ ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှုအတွက် Health အက်ပ်နှင့် ပြီးပြည့်စုံသော sync၊ widget ပံ့ပိုးမှုနှင့် ရင်းနှီးသော Apple ဒီဇိုင်း ဘာသာစကား။

🔬

သုတေသန-အခြေခံ မက်ထရစ်များ

မက်ထရစ်အားလုံး peer-reviewed သိပ္ပံ သုတေသန အပေါ် အခြေခံသည်။ Wakayoshi et al. မှ CRS၊ IF² ဖော်မြူလာဖြင့် ပြေးခြင်းအတွက် လိုက်လျောညီထွေဖြစ်အောင် ပြုလုပ်ထားသော TSS၊ သက်သေပြထားသော CTL/ATL မော်ဒယ်များ။

👥

နည်းပြ-ဖော်ရွေ့ အစီရင်ခံစာများ

နည်းပြများအတွက် အသေးစိတ် အစီရင်ခံစာများ တင်ပို့ပါ။ အီးမေးလ်ဖြင့် HTML အကျဉ်းချုပ်များ မျှဝေပါ၊ spreadsheet ခွဲခြမ်းစိတ်ဖြာမှုအတွက် CSV သို့မဟုတ် လေ့ကျင့်ရေး မှတ်တမ်းများနှင့် မှတ်တမ်းများအတွက် PDF။

🌍

နေရာတိုင်းတွင် အလုပ်လုပ်သည်

လမ်းကြောင်း သို့မဟုတ် တောင်တန်း၊ sprint သို့မဟုတ် အကွာအဝေး။ Run Analytics သည် ပြေးခြင်း ထိရောက်မှု မက်ထရစ်များကို ပြေးခြင်း အမျိုးအစားအားလုံးနှင့် လိုက်လျောညီထွေဖြစ်အောင် ပြုလုပ်ပြီး လေ့ကျင့်မှု လက္ခဏာများကို အလိုအလျောက် ရှာဖွေသည်။

အမြဲ တိုးတက်နေသည်

အသုံးပြုသူ တုံ့ပြန်ချက် အပေါ် အခြေခံ၍ ပြေးခြင်း စွမ်းဆောင်ရည် အင်္ဂါရပ်အသစ်များဖြင့် ပုံမှန် အပ်ဒိတ်များ။ မကြာသေးမီက ထည့်သွင်းမှုများတွင် နှစ်စဉ် နှိုင်းယှဉ်မှုများ၊ ကိုယ်ပိုင် မှတ်တမ်း ခြေရာခံခြင်းနှင့် တိုးမြှင့်ထားသော တင်ပို့မှု ရွေးချယ်စရာများ ပါဝင်သည်။

မေးလေ့ရှိသော မေးခွန်းများ

မကြာခဏ မေးလေ့ရှိသော မေးခွန်းများ

ဤ ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှု အက်ပ်သည် ကျွန်ုပ်၏ ဒေတာကို မည်သို့ ရရှိသနည်း။

Run Analytics သည် Apple Health နှင့် sync လုပ်၍ တွဲဖက်အသုံးပြုနိုင်သော စက် သို့မဟုတ် အက်ပ်မှ မှတ်တမ်းတင်ထားသော ပြေးခြင်း လေ့ကျင့်မှုများကို တင်သွင်းသည်။ ၎င်းတွင် smart watch များ၊ fitness tracker များနှင့် လက်ဖြင့် ထည့်သွင်းမှုများ ပါဝင်သည်။ အက်ပ်သည် ဤဒေတာကို ပြည်တွင်းတွင် လုပ်ဆောင်၍ အဆင့်မြင့် ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်များကို တွက်ချက်သည်။

Critical running speed စမ်းသပ်မှုဆိုသည်မှာ အဘယ်နည်း၊ မည်သို့ လုပ်ဆောင်ရမည်နည်း။

Critical running speed (CRS) သည် အများဆုံး ကြိုးစားမှု ပြေးခြင်း ၂ ခုကို အသုံးပြုသော သိပ္ပံနည်းကျ ပရိုတိုကောဖြစ်သည်: 5K နှင့် 3K ကြားတွင် ၁၀-၂၀ မိနစ် နားခြင်းဖြင့်။ အက်ပ်သည် ဤအချိန်များမှ သင့် aerobic threshold ကို တွက်ချက်ပြီး ပြေးခြင်း လေ့ကျင့်ရေး ဇုန်အားလုံးကို အလိုအလျောက် ချိန်ညှိသည်။ တိုးတက်မှုကို ခြေရာခံရန် ၆-၈ ပတ်တိုင်း ထပ်လုပ်ပါ။

ကျွန်ုပ်၏ ပြေးခြင်း ဒေတာကို cloud သို့ တင်သနည်း။

မဟုတ်ပါ။ Run Analytics သည် ပြေးခြင်း ဒေတာအားလုံးကို သင့် iPhone တွင် ပြည်တွင်း၌ လုပ်ဆောင်သည်။ ပြင်ပ ဆာဗာမရှိ၊ cloud အကောင့်မရှိ၊ ဒေတာ လွှဲပြောင်းမှုမရှိ။ တင်ပို့မှုများကို သင် ထိန်းချုပ်သည်: သင့် ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်များဖြင့် JSON၊ CSV၊ HTML သို့မဟုတ် PDF ဖိုင်များ ဖန်တီးပြီး သင်လိုချင်သလို မျှဝေပါ။

ဤ ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှု အက်ပ်ကို တောင်တန်း ပြေးခြင်းအတွက် အသုံးပြုနိုင်သလား။

ဟုတ်ကဲ့။ Run Analytics သည် တောင်တန်း ပြေးခြင်း အပါအဝင် Apple Health ရှိ မည်သည့် ပြေးခြင်း လေ့ကျင့်မှုနှင့်မဆို အလုပ်လုပ်သည်။ အက်ပ်သည် ပြေးခြင်း ထိရောက်မှု မက်ထရစ်များကို ရရှိနိုင်သော ဒေတာနှင့် လိုက်လျောညီထွေဖြစ်အောင် ပြုလုပ်ပြီး လမ်းကြောင်း သို့မဟုတ် တောင်တန်းတွင် ပတ်ဝန်းကျင်တစ်ခုစီအတွက် သက်ဆိုင်ရာ စွမ်းဆောင်ရည် ခွဲခြမ်းစိတ်ဖြာမှု ပေးသည်။

လစဉ်နှင့် နှစ်စဉ် အစီအစဉ်များ ကွာခြားချက်က ဘာလဲ။

အစီအစဉ် နှစ်ခုလုံးသည် တူညီသော အင်္ဂါရပ်များ ပေးသည်: ပြေးခြင်း စွမ်းဆောင်ရည် မက်ထရစ်အားလုံး၊ အကန့်အသတ်မရှိ လေ့ကျင့်ရေး ဇုန်များ၊ အချိန်ကာလ နှိုင်းယှဉ်မှုများ၊ တင်ပို့မှု အမျိုးမျိုးနှင့် အခမဲ့ အပ်ဒိတ်များ။ ကွာခြားချက်တစ်ခုတည်းမှာ စျေးနှုန်းဖြစ်သည်: နှစ်စဉ်သည် 18% သက်သာသည် (တစ်လလျှင် €3.99 နှင့် နှိုင်းယှဉ်လျှင် €3.25)။

ကျွန်ုပ်၏ subscription ကို အချိန်မရွေး ပယ်ဖျက်နိုင်သလား။

ဟုတ်ကဲ့။ Subscription များကို App Store မှတဆင့် စီမံခန့်ခွဲသောကြောင့် Settings → [သင့်အမည်] → Subscriptions မှ အချိန်မရွေး ပယ်ဖျက်နိုင်သည်။ ပယ်ဖျက်ပါက သင့် လက်ရှိ ငွေပေးချေမှု ကာလ ကုန်ဆုံးသည်အထိ ဝင်ရောက်ခွင့် ဆက်လက် ရရှိမည်ဖြစ်သည်။

သင့် ပြေးခြင်း စွမ်းဆောင်ရည်ကို ပြောင်းလဲရန် အသင့်ဖြစ်ပြီလား။

စွမ်းဆောင်ရည် တိုးတက်စေရန် သိပ္ပံနည်းကျ ပြေးခြင်း ခွဲခြမ်းစိတ်ဖြာမှု အသုံးပြုနေသော ပြေးသမား ထောင်ပေါင်းများစွာနှင့် ပူးပေါင်းပါ။ ဤ ကိုယ်ရေးကိုယ်တာ-ဦးစားပေး iOS ပြေးခြင်း အက်ပ်၏ ၇ ရက် အခမဲ့ စမ်းသုံးခွင့်ကို ယနေ့ စတင်ပါ။

အကောင့် မလိုအပ်ပါ • အချိန်မရွေး ပယ်ဖျက်နိုင်သည် • သင့်ဒေတာ သင့်စက်တွင် ရှိနေသည်
