隐私政策

最后更新：2025年10月27日

您的隐私受到保护

Run Analytics不收集、存储或传输您的任何个人数据。所有数据处理都在您的设备上本地进行。

概述

Run Analytics以您的隐私为首要优先考虑而设计。本隐私政策解释了我们对保护您个人信息和数据的承诺。

简而言之：我们不收集您的数据，因为我们不需要。您在应用中所做的一切都保留在您的设备上。

我们不收集的信息

Run Analytics不收集、传输或存储：

  • 个人信息（姓名、电子邮件、电话号码等）
  • 跑步活动数据（距离、配速、路线、时间等）
  • 健康数据（心率、卡路里等）
  • 位置数据或GPS坐标
  • 设备信息或标识符
  • 使用统计或分析
  • 崩溃报告或诊断
  • 任何其他个人或敏感信息

应用如何工作

Run Analytics中的所有计算和数据处理都完全在您的设备上进行：

  • 您的跑步数据仅存储在设备的本地存储中
  • 所有计算（配速、分段、区间等）都在本地执行
  • 没有数据发送到我们的服务器或任何第三方服务
  • 应用功能不需要互联网连接

数据存储

您在应用中输入或生成的任何数据都专门使用iOS本地存储机制存储在您的设备上。这些数据始终在您的完全控制下：

  • 您的数据始终保留在您的设备上
  • 我们无法访问您的数据
  • 如果您删除应用，所有数据将从您的设备上永久删除
  • 除非您明确启用iOS应用备份，否则数据不会同步到iCloud或任何云服务

第三方服务

Run Analytics不使用任何第三方服务，包括：

  • 无分析服务（Google Analytics、Firebase等）
  • 无广告网络
  • 无崩溃报告服务
  • 无社交媒体集成
  • 无支付处理器（应用免费）

权限

只有当您选择使用特定功能时，应用才可能请求以下iOS权限：

  • HealthKit/健康应用：仅当您选择导入锻炼数据时。所有数据保留在您的设备上。
  • 通知：仅当您启用锻炼提醒时。没有数据发送到设备外。

所有权限都是可选的，您可以随时通过iOS设置撤销它们。

儿童隐私

因为我们不收集任何数据，Run Analytics对所有年龄的用户都是安全的，包括13岁以下的儿童。但是，我们建议所有健身活动都在家长指导下进行。

隐私政策变更

我们可能会不时更新此隐私政策，以反映应用或法律要求的变化。任何变更都将发布在此页面上，并更新"最后更新"日期。

重要：我们永远不会改变不收集您数据的核心承诺。

联系我们

如果您对此隐私政策或应用有任何问题，请通过我们的联系页面联系我们。

总结

Run Analytics是一个隐私优先的应用。我们相信您的数据属于您，应该与您在一起。

  • ❌ 无数据收集
  • ❌ 无跟踪或分析
  • ❌ 无第三方服务
  • ✅ 100%本地处理
  • ✅ 完全控制您的数据
