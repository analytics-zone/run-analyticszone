Связаться с Run Analytics

Мы будем рады услышать вас! Если у вас есть вопросы о беговой аналитике, нужна помощь с тестированием CRS, вы хотите сообщить об ошибке или предложить новые функции — мы здесь, чтобы помочь.

Получить поддержку и поделиться отзывом

Команда Run Analytics стремится помочь соревнующимся бегунам и триатлетам извлечь максимум из своих тренировочных данных. Обычно мы отвечаем на все запросы в течение 24-48 часов в рабочие дни.

Прежде чем связаться с нами

Ознакомьтесь с нашими подробными руководствами для ответов на распространенные вопросы:

  • Руководство по началу работы — Полное введение и обучение CRS-тесту
  • Руководство по калькулятору CRS — Понимание критической скорости бега
  • Руководство по TSS — Объяснение показателя тренировочного стресса
  • Тренировочные зоны — Разбор 5-зонной системы
  • Научные исследования — Научные основы с рецензированием
