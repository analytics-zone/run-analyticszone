Preguntas Frecuentes

¿Cómo obtiene mis datos esta aplicación de análisis de ejecución?

Run Analytics sincroniza con Apple Health para importar entrenamientos de ejecución grabados por cualquier Compatible con dispositivos o aplicaciones. Esto incluye relojes inteligentes, monitores de actividad y entradas manuales. La aplicación procesa estos datos localmente para calcular métricas avanzadas de rendimiento de funcionamiento.

¿Qué es el test de velocidad crítica de carrera y cómo lo hago?

La velocidad crítica de carrera (CRS) es un protocolo científico utilizando 2 carreras de esfuerzo máximo: 5K y 3K con 10-20 minutos de descanso entre ellas. La aplicación calcula tu umbral aeróbico a partir de estos tiempos y ajusta automáticamente todas las zonas de entrenamiento. Repite cada 6-8 semanas para seguir el progreso.

¿Mis datos de running se suben a la nube?

No. Run Analytics procesa todos los datos de ejecución localmente en tu iPhone. No hay servidores externos, ni cuentas en la nube, ni transferencias de datos. Tú controlas las exportaciones: géneros archivos JSON, CSV, HTML o PDF con tus métricas de rendimiento y compártelos como quieras.

¿Puedo usar esta aplicación de análisis de running para trail running?

Si. Run Analytics funciona con cualquier entrenamiento de carrera en Apple Health, incluido el trail running. La aplicación se adapta las métricas de eficiencia a los datos disponibles ya sea en pista o trail, proporcionando análisis de rendimiento relevante para cada entorno.

¿Qué diferencia hay entre los aviones mensuales y anuales?

Ambos planos ofrecen funciones idénticas: todas las métricas de rendimiento, zonas de entrenamiento comparaciones ilimitadas temporales, múltiples exportaciones y actualizaciones gratuitas. La única diferencia es el precio: el anual. ahorra un 18% (equivalente a 3,25€/mes vs 3,99€/mes).

¿Puedo cancelar la suscripción en cualquier momento?

Si. Las suscripciones se gestionan a través del App Store, así que puedes cancelar cuando quieras desde Configuración → [Tu nombre] → Suscripciones. Si cancelas, mantendrás el acceso hasta el final de tu período de facturación actual.
