Privaatheid Beleid

Laaste opgedateer: Oktober 27, 2025

Jou Privaatheid is Beskerm

Run Analytics versamel, stoor, of dra nie enige van jou persoonlike data oor nie. Alle data verwerking gebeur plaaslik op jou toestel.

Oorsig

Run Analytics is ontwerp met jou privaatheid as 'n top prioriteit. Hierdie Privaatheid Beleid verduidelik ons verbintenis tot die beskerming van jou persoonlike inligting en data.

Kortliks: Ons versamel nie jou data nie omdat ons dit nie nodig het nie. Alles wat jy in die toepassing doen bly op jou toestel.

Inligting Wat Ons NIE Versamel Nie

Run Analytics versamel, dra oor, of stoor nie:

  • Persoonlike inligting (naam, e-pos, telefoonnommer, ens.)
  • Hardloop aktiwiteit data (afstand, tempo, roetes, tye, ens.)
  • Gesondheid data (harttempo, kalorieë, ens.)
  • Ligging data of GPS koördinate
  • Toestel inligting of identifiseerders
  • Gebruiks statistieke of analise
  • Ongeluk verslae of diagnostiek
  • Enige ander persoonlike of sensitiewe inligting

Hoe die Toepassing Werk

Alle berekeninge en data verwerking in Run Analytics gebeur heeltemal op jou toestel:

  • Jou hardloop data word slegs in jou toestel se plaaslike berging gestoor
  • Alle berekeninge (tempo, splitte, sones, ens.) word plaaslik uitgevoer
  • Geen data word ooit na ons bedieners of enige derde-party dienste gestuur nie
  • Geen internet verbinding word vereis vir die toepassing om te funksioneer nie

Data Berging

Enige data wat jy invoer of genereer in die toepassing word uitsluitlik op jou toestel gestoor deur gebruik te maak van iOS plaaslike berging meganismes. Hierdie data bly onder jou volledige beheer:

  • Jou data bly te alle tye op jou toestel
  • Ons kan nie jou data toegang nie
  • As jy die toepassing uitvee, word alle data permanent van jou toestel verwyder
  • Data word nie gesinkroniseer na iCloud of enige wolk diens nie tensy jy uitdruklik iOS toepassing rugsteun aktiveer

Derde-Party Dienste

Run Analytics gebruik nie enige derde-party dienste nie, insluitend:

  • Geen analise dienste (Google Analytics, Firebase, ens.)
  • Geen advertensie netwerke
  • Geen ongeluk rapportering dienste
  • Geen sosiale media integrasies
  • Geen betaling verwerkers (die toepassing is gratis)

Toestemmings

Die toepassing kan die volgende iOS toestemmings versoek slegs as jy kies om spesifieke kenmerke te gebruik:

  • HealthKit/Health Toepassing: Slegs as jy kies om oefensessie data in te voer. Alle data bly op jou toestel.
  • Kennisgewings: Slegs as jy oefensessie herinneringe aktiveer. Geen data word buite jou toestel gestuur nie.

Alle toestemmings is opsioneel en jy kan hulle te enige tyd herroep deur iOS Instellings.

Kinders se Privaatheid

Omdat ons geen data versamel nie, is Run Analytics veilig vir gebruikers van alle ouderdomme, insluitend kinders onder 13. Ons beveel egter ouerlike leiding aan vir alle fiksheid aktiwiteite.

Veranderinge aan Hierdie Privaatheid Beleid

Ons kan hierdie Privaatheid Beleid van tyd tot tyd opdateer om veranderinge in die toepassing of wetlike vereistes te weerspieël. Enige veranderinge sal op hierdie bladsy geplaas word met 'n opgedateerde "Laaste opgedateer" datum.

Belangrik: Ons sal nooit ons kern verbintenis verander om nie jou data te versamel nie.

Kontak Ons

As jy enige vrae het oor hierdie Privaatheid Beleid of die toepassing, kontak ons asseblief deur ons kontak bladsy.

Opsomming

Run Analytics is 'n privaatheid-eerste toepassing. Ons glo jou data behoort aan jou en moet by jou bly.

  • ❌ Geen data versameling
  • ❌ Geen opsporing of analise
  • ❌ Geen derde-party dienste
  • ✅ 100% plaaslike verwerking
  • ✅ Volledige beheer oor jou data
