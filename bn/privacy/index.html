গোপনীয়তা নীতি

(আইনি পাঠ্যের বাংলা অনুবাদ)

সর্বশেষ আপডেট: ১০ জানুয়ারি, ২০২৫ | কার্যকরী তারিখ: ১০ জানুয়ারি, ২০২৫

ভূমিকা

Run Analytics ("আমরা," "আমাদের," বা "অ্যাপ") আপনার গোপনীয়তা রক্ষায় প্রতিশ্রুতিবদ্ধ। এই গোপনীয়তা নীতি ব্যাখ্যা করে যে কীভাবে আমাদের মোবাইল অ্যাপ্লিকেশনগুলি (iOS এবং Android) আপনার ডিভাইস থেকে স্বাস্থ্য ডেটা অ্যাক্সেস, ব্যবহার এবং সুরক্ষা করে।

গোপনীয়তার মূল নীতি: Run Analytics একটি জিরো-সার্ভার, লোকাল-অনলি আর্কিটেকচার-এ কাজ করে। Apple HealthKit (iOS) বা Health Connect (Android) থেকে অ্যাক্সেস করা সমস্ত স্বাস্থ্য ডেটা একচেটিয়াভাবে আপনার ফিজিক্যাল ডিভাইসেই থাকে এবং কখনও কোনো এক্সটার্নাল সার্ভার, ক্লাউড পরিষেবা বা তৃতীয় পক্ষের কাছে পাঠানো হয় না।

১. স্বাস্থ্য ডেটা অ্যাক্সেস

Run Analytics বিস্তারিত রানিং ওয়ার্কআউট বিশ্লেষণ প্রদান করতে আপনার ডিভাইসের নেটিভ স্বাস্থ্য প্ল্যাটফর্মের সাথে ইন্টিগ্রেট করে:

১.১ iOS - Apple HealthKit ইন্টিগ্রেশন

iOS ডিভাইসে, Run Analytics রানিং ওয়ার্কআউট ডেটা অ্যাক্সেস করতে Apple HealthKit-এর সাথে ইন্টিগ্রেট করে। আমরা শুধুমাত্র রিড-অনলি অ্যাক্সেস অনুরোধ করি:

  • ওয়ার্কআউট সেশন: সময় এবং সময়কাল সহ রানিং এক্সারসাইজ সেশন
  • দূরত্ব: মোট রানিং দূরত্ব
  • হার্ট রেট: ওয়ার্কআউটের সময় হার্ট রেট ডেটা
  • অ্যাক্টিভ এনার্জি: রানিং সেশনের সময় পোড়ানো ক্যালোরি
  • স্টেপ কাউন্ট: রানিংয়র সময় নেওয়া পদক্ষেপের বিশ্লেষণ
  • রানিং স্পিড: পেস এবং স্পিড বিশ্লেষণের জন্য মেট্রিক্স

Apple HealthKit কমপ্লায়েন্স: Run Analytics সমস্ত Apple HealthKit নির্দেশিকা মেনে চলে। আপনার স্বাস্থ্য ডেটা পুরোপুরি আপনার iOS ডিভাইসে প্রসেস করা হয় এবং কখনও এটি ছেড়ে যায় না। আমরা কখনই তৃতীয় পক্ষ, বিজ্ঞাপন প্ল্যাটফর্ম বা ডেটা ব্রোকারদের সাথে HealthKit ডেটা শেয়ার করি না।

১.২ Android - Health Connect ইন্টিগ্রেশন

স্বাস্থ্য ডেটার ধরনঅনুমতিউদ্দেশ্য
এক্সারসাইজ সেশনREAD_EXERCISEHealth Connect থেকে রানিং ওয়ার্কআউট সেশন শনাক্ত এবং ইম্পোর্ট করতে
দূরত্ব রেকর্ডREAD_DISTANCEপ্রতি রানের মোট দূরত্ব প্রদর্শন এবং পেস গণনা করতে
হার্ট রেট রেকর্ডREAD_HEART_RATEহার্ট রেট চার্ট প্রদর্শন, গড় এবং সর্বোচ্চ হার্ট রেট গণনা করতে
স্পিড রেকর্ডREAD_SPEEDআপনার রানিং স্পিড এবং পেস জোন গণনা এবং প্রদর্শন করতে
স্টেপসREAD_STEPSরানিং সেশনের সময় পদক্ষেপ গণনা করতে
পোড়ানো ক্যালোরিREAD_TOTAL_CALORIES_BURNEDরানের সময় শক্তি ব্যয়ের ওভারভিউ প্রদান করতে

১.৩ আমরা কীভাবে স্বাস্থ্য ডেটা ব্যবহার করি

সমস্ত স্বাস্থ্য ডেটা একচেটিয়াভাবে নিম্নলিখিত উদ্দেশ্যে ব্যবহার করা হয়:

  • ওয়ার্কআউট প্রদর্শন: বিস্তারিত মেট্রিক্স (দূরত্ব, সময়, পেস, হার্ট রেট) সহ আপনার রানিং সেশন দেখানো
  • পারফরম্যান্স অ্যানালিটিক্স: ক্রিটিক্যাল রানিং স্পিড (CRS), ট্রেনিং স্ট্রেস স্কোর (rTSS), এবং ট্রেনিং জোন গণনা করা
  • অগ্রগতি ট্র্যাকিং: পারফরম্যান্স ট্রেন্ড, ব্যক্তিগত সেরা এবং ওয়ার্কআউট সারাংশ প্রদর্শন করা
  • ডেটা এক্সপোর্ট: ব্যক্তিগত ব্যবহারের জন্য আপনাকে আপনার ওয়ার্কআউট ডেটা CSV, JSON, বা PDF ফরম্যাটে এক্সপোর্ট করতে দেওয়া

১.৪ ডেটা স্টোরেজ

🔒 গুরত্বপূর্ণ গোপনীয়তা গ্যারান্টি:

সমস্ত স্বাস্থ্য ডেটা একচেটিয়াভাবে আপনার ফিজিক্যাল ডিভাইসে থাকে।

  • iOS: ডেটা iOS Core Data এবং UserDefaults ব্যবহার করে সংরক্ষণ করা হয় (শুধুমাত্র অন-ডিভাইস)
  • Android: ডেটা Android Room Database ব্যবহার করে সংরক্ষণ করা হয় (অন-ডিভাইস SQLite)
  • কোনো ডেটা এক্সটার্নাল সার্ভারে আপলোড করা হয় না
  • ইন্টারনেটের মাধ্যমে কোনো ডেটা পাঠানো হয় না
  • স্বাস্থ্য ডেটার কোনো ক্লাউড সিঙ্ক্রোনাইজেশন বা ব্যাকআপ নেই
  • আপনার স্বাস্থ্য ডেটায় কোনো তৃতীয় পক্ষের অ্যাক্সেস নেই

ডেটা কেবল তখনই আপনার ডিভাইস থেকে বের হয় যখন আপনি স্পষ্টভাবে নির্বাচন করেন আপনার ওয়ার্কআউটগুলি এক্সপোর্ট করতে এবং ফাইলটি নিজে শেয়ার করতে।

২. প্রয়োজনীয় অনুমতিসমূহ

২.১ iOS অনুমতিসমূহ

  • HealthKit অ্যাক্সেস: ওয়ার্কআউট, দূরত্ব, হার্ট রেট, অ্যাক্টিভ এনার্জি, এবং রানিং স্পিড-এর রিড অ্যাক্সেস
  • ফটো লাইব্রেরি (ঐচ্ছিক): শুধুমাত্র যদি আপনি ওয়ার্কআউট সারাংশ ইমেজ হিসেবে সেভ করতে চান

আপনি যেকোনো সময় iOS Settings → Privacy & Security → Health → Run Analytics-এ গিয়ে HealthKit অনুমতি পরিচালনা করতে পারেন।

২.২ Android অনুমতিসমূহ

  • android.permission.health.READ_EXERCISE
  • android.permission.health.READ_DISTANCE
  • android.permission.health.READ_HEART_RATE
  • android.permission.health.READ_SPEED
  • android.permission.health.READ_STEPS
  • android.permission.health.READ_TOTAL_CALORIES_BURNED
  • ইন্টারনেট অ্যাক্সেস (INTERNET): শুধুমাত্র অ্যাপের মধ্যে স্ট্যাটিক কন্টেন্ট প্রদর্শনের জন্য ব্যবহৃত হয়।

৩. যে ডেটা আমরা সংগ্রহ করি না

Run Analytics নাম, ইমেল, অবস্থান বা ব্যবহারের অ্যানালিটিক্স সংগ্রহ করে না।

৪. ইন-অ্যাপ ক্রয়

সাবস্ক্রিপশনগুলি App Store/Google Play দ্বারা পরিচালিত হয়। আমরা কখনই পেমেন্টের বিবরণ দেখি না।

৫. ডেটা সংরক্ষণ

ডেটা অনির্দিষ্টকালের জন্য ডিভাইসে থাকে যতক্ষণ না আপনি এটি ডিলিট করেন বা অ্যাপটি আনইনস্টল করেন।
