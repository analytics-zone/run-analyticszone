শর্তাবলী

(আইনি পাঠ্যের বাংলা অনুবাদ)

সর্বশেষ আপডেট: ১০ জানুয়ারি, ২০২৫

১. ভূমিকা

এই শর্তাবলী ("Terms") আপনার Run Analytics মোবাইল অ্যাপ্লিকেশনটির ("অ্যাপ") ব্যবহার নিয়ন্ত্রণ করে। অ্যাপটি ডাউনলোড, ইনস্টল বা ব্যবহার করে, আপনি এই শর্তাবলী দ্বারা আবদ্ধ হতে সম্মত হন। যদি আপনি এই শর্তাবলীতে সম্মত না হন, তবে অনুগ্রহ করে অ্যাপটি ব্যবহার করবেন না।

২. ব্যবহারের লাইসেন্স

Run Analytics আপনাকে আপনার মালিকানাধীন বা নিয়ন্ত্রিত ডিভাইসগুলিতে আপনার ব্যক্তিগত, অবাণিজ্যিক উদ্দেশ্যে অ্যাপটি ব্যবহার করার জন্য একটি সীমিত, নন-এক্সক্লুসিভ, নন-ট্রান্সফারযোগ্য, প্রত্যাহারযোগ্য লাইসেন্স প্রদান করে, যা এই শর্তাবলী এবং অ্যাপ স্টোরের প্রযোজ্য নিয়ম সাপেক্ষে।

৩. মেডিকেল ডিসক্লেইমার

গুরুত্বপূর্ণ: এটি কোনো চিকিৎসা পরামর্শ নয়

Run Analytics একটি ফিটনেস ট্র্যাকিং এবং বিশ্লেষণ টুল, কোনো মেডিকেল ডিভাইস নয়। অ্যাপ দ্বারা প্রদত্ত ডেটা, মেট্রিক্স এবং অন্তর্দৃষ্টি (হার্ট রেট বিশ্লেষণ, স্টেপ কাউন্ট এবং অ্যাক্টিভিটি স্কোর সহ) শুধুমাত্র তথ্যগত উদ্দেশ্যে।

  • যেকোনো নতুন এক্সারসাইজ প্রোগ্রাম শুরু করার আগে সর্বদা চিকিৎসকের সাথে পরামর্শ করুন।
  • কোনো স্বাস্থ্য অবস্থা নির্ণয় বা চিকিৎসার জন্য অ্যাপের উপর নির্ভর করবেন না।
  • হাঁটার সময় ব্যথা, মাথা ঘোরা বা শ্বাসকষ্ট অনুভব করলে অবিলম্বে থামুন এবং চিকিৎসার সাহায্য নিন।

৪. ডেটা গোপনীয়তা

আপনার গোপনীয়তা আমাদের কাছে সর্বাগ্রে। আমাদের গোপনীয়তা নীতিতে বর্ণিত হিসাবে, Run Analytics একটি লোকাল-অনলি আর্কিটেকচারে কাজ করে। আমরা আমাদের সার্ভারে আপনার স্বাস্থ্য ডেটা সংরক্ষণ করি না। আপনি আপনার ডিভাইসে আপনার ডেটার সম্পূর্ণ মালিকানা এবং নিয়ন্ত্রণ বজায় রাখেন।

৫. সাবস্ক্রিপশন এবং পেমেন্ট

Run Analytics ইন-অ্যাপ সাবস্ক্রিপশনের মাধ্যমে প্রিমিয়াম বৈশিষ্ট্য অফার করতে পারে ("Pro Mode")।

  • পেমেন্ট প্রসেসিং: সমস্ত পেমেন্ট Apple (iOS-এর জন্য) বা Google (Android-এর জন্য) দ্বারা নিরাপদে প্রসেস করা হয়। আমরা আপনার পেমেন্ট তথ্য সংরক্ষণ করি না।
  • অটো-রিনিউয়াল: বর্তমান সময়কাল শেষ হওয়ার কমপক্ষে ২৪ ঘন্টা আগে বন্ধ না করা হলে সাবস্ক্রিপশনগুলি স্বয়ংক্রিয়ভাবে রিনিউ হয়।
  • বাতিলকরণ: আপনি আপনার ডিভাইস সেটিংসে সাবস্ক্রিপশন পরিচালনা এবং বাতিল করতে পারেন।
  • রিফান্ড: রিফান্ড অনুরোধগুলি Apple বা Google দ্বারা তাদের নিজ নিজ রিফান্ড নীতি অনুযায়ী পরিচালিত হয়।

৬. মেধাস্বত্ব

অ্যাপটি, এর কোড, ডিজাইন, গ্রাফিক্স এবং অ্যালগরিদম (যেমন অ্যাক্টিভিটি স্কোর এবং স্টেপ বিশ্লেষণের নির্দিষ্ট বাস্তবায়ন) Run Analytics-এর মেধাস্বত্ব এবং কপিরাইট আইন দ্বারা সুরক্ষিত। আপনি অ্যাপের সোর্স কোড রিভার্স ইঞ্জিনিয়ার, ডিকম্পাইল বা কপি করতে পারবেন না।

৭. দায়বদ্ধতার সীমাবদ্ধতা

আইন অনুযায়ী সর্বোচ্চ সীমা পর্যন্ত, Run Analytics আপনার অ্যাপ ব্যবহারের ফলে কোনো পরোক্ষ, আনুষঙ্গিক, বিশেষ, পরিণতিমূলক বা শাস্তিমূলক ক্ষতির জন্য দায়ী থাকবে না। অ্যাপটি কোনো ওয়ারেন্টি ছাড়াই "যেমন আছে" তেমন প্রদান করা হয়।

৮. শর্তাবলীর পরিবর্তন

আমরা যেকোনো সময় এই শর্তাবলী পরিবর্তন করার অধিকার রাখি। আমরা "সর্বশেষ আপডেট" তারিখ আপডেট করে আপনাকে কোনো পরিবর্তনের বিষয়ে অবহিত করব। পরিবর্তনের পরে অ্যাপের ক্রমাগত ব্যবহার নতুন শর্তাবলীর গ্রহণযোগ্যতা হিসাবে গণ্য হবে।
