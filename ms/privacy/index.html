Dasar Privasi

Kemas kini terakhir: 27 Oktober 2025

Privasi Anda Dilindungi

Run Analytics tidak mengumpul, menyimpan, atau menghantar sebarang data peribadi anda. Semua pemprosesan data berlaku secara tempatan pada peranti anda.

Gambaran Keseluruhan

Run Analytics direka dengan privasi anda sebagai keutamaan utama. Dasar Privasi ini menerangkan komitmen kami untuk melindungi maklumat dan data peribadi anda.

Ringkasnya: Kami tidak mengumpul data anda kerana kami tidak memerlukannya. Semua yang anda lakukan dalam aplikasi ini kekal pada peranti anda.

Maklumat Yang TIDAK Kami Kumpul

Run Analytics tidak mengumpul, menghantar, atau menyimpan:

  • Maklumat peribadi (nama, e-mel, nombor telefon, dll.)
  • Data aktiviti larian (jarak, kadar, laluan, masa, dll.)
  • Data kesihatan (kadar denyutan jantung, kalori, dll.)
  • Data lokasi atau koordinat GPS
  • Maklumat atau pengecam peranti
  • Statistik penggunaan atau analitik
  • Laporan ranap atau diagnostik
  • Sebarang maklumat peribadi atau sensitif yang lain

Cara Aplikasi Berfungsi

Semua pengiraan dan pemprosesan data dalam Run Analytics berlaku sepenuhnya pada peranti anda:

  • Data larian anda disimpan hanya dalam storan tempatan peranti anda
  • Semua pengiraan (kadar, pecahan, zon, dll.) dilakukan secara tempatan
  • Tiada data dihantar ke pelayan kami atau mana-mana perkhidmatan pihak ketiga
  • Tiada sambungan internet diperlukan untuk aplikasi berfungsi

Penyimpanan Data

Sebarang data yang anda masukkan atau jana dalam aplikasi disimpan secara eksklusif pada peranti anda menggunakan mekanisme penyimpanan tempatan iOS. Data ini kekal di bawah kawalan penuh anda:

  • Data anda kekal pada peranti anda pada setiap masa
  • Kami tidak boleh mengakses data anda
  • Jika anda memadam aplikasi, semua data dikeluarkan secara kekal daripada peranti anda
  • Data tidak diselaraskan ke iCloud atau mana-mana perkhidmatan awan melainkan anda secara eksplisit mengaktifkan sandaran aplikasi iOS

Perkhidmatan Pihak Ketiga

Run Analytics tidak menggunakan sebarang perkhidmatan pihak ketiga, termasuk:

  • Tiada perkhidmatan analitik (Google Analytics, Firebase, dll.)
  • Tiada rangkaian pengiklanan
  • Tiada perkhidmatan pelaporan ranap
  • Tiada integrasi media sosial
  • Tiada pemproses pembayaran (aplikasi ini percuma)

Kebenaran

Aplikasi mungkin meminta kebenaran iOS berikut hanya jika anda memilih untuk menggunakan ciri tertentu:

  • HealthKit/Aplikasi Kesihatan: Hanya jika anda memilih untuk mengimport data senaman. Semua data kekal pada peranti anda.
  • Notifikasi: Hanya jika anda mengaktifkan peringatan senaman. Tiada data dihantar keluar daripada peranti anda.

Semua kebenaran adalah pilihan dan anda boleh membatalkannya pada bila-bila masa melalui Tetapan iOS.

Privasi Kanak-Kanak

Oleh kerana kami tidak mengumpul sebarang data, Run Analytics adalah selamat untuk pengguna semua peringkat umur, termasuk kanak-kanak bawah umur 13 tahun. Walau bagaimanapun, kami mengesyorkan bimbingan ibu bapa untuk semua aktiviti kecergasan.

Perubahan pada Dasar Privasi Ini

Kami mungkin mengemaskini Dasar Privasi ini dari semasa ke semasa untuk mencerminkan perubahan dalam aplikasi atau keperluan undang-undang. Sebarang perubahan akan disiarkan pada halaman ini dengan tarikh "Kemas kini terakhir" yang dikemaskini.

Penting: Kami tidak akan pernah mengubah komitmen teras kami untuk tidak mengumpul data anda.

Hubungi Kami

Jika anda mempunyai sebarang soalan mengenai Dasar Privasi ini atau aplikasi ini, sila hubungi kami melalui halaman hubungi kami.

Ringkasan

Run Analytics adalah aplikasi yang mengutamakan privasi. Kami percaya data anda adalah milik anda dan harus kekal bersama anda.

  • ❌ Tiada pengumpulan data
  • ❌ Tiada penjejakan atau analitik
  • ❌ Tiada perkhidmatan pihak ketiga
  • ✅ 100% pemprosesan tempatan
  • ✅ Kawalan penuh terhadap data anda
