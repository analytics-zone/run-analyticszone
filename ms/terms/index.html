Terma dan Syarat

Kemaskini terakhir: Januari 2024

Pengenalan

Terma dan Syarat ini mengawal penggunaan anda terhadap laman web kami. Dengan mengakses atau menggunakan laman web kami, anda bersetuju untuk terikat dengan terma ini.

Jika anda tidak bersetuju dengan mana-mana bahagian terma ini, anda tidak sepatutnya menggunakan laman web kami.

Penggunaan Laman Web

Anda bersetuju untuk:

  • Menggunakan laman web untuk tujuan yang sah sahaja
  • Tidak cuba mendapatkan akses tanpa kebenaran kepada mana-mana bahagian laman web
  • Tidak mengganggu fungsi yang betul laman web
  • Tidak menghantar sebarang kod yang membahayakan atau berniat jahat
  • Menghormati hak harta intelek orang lain

Harta Intelek

Semua kandungan di laman web ini, termasuk teks, grafik, logo, imej, dan perisian, adalah hak milik pemilik laman web atau pemberi lesennya dan dilindungi oleh hak cipta dan undang-undang harta intelek lain.

Anda tidak boleh menghasilkan semula, mengedarkan, mengubah suai, atau mencipta karya terbitan dari sebarang kandungan di laman web ini tanpa kebenaran bertulis terlebih dahulu.

Penafian Jaminan

Laman web ini disediakan "sebagaimana adanya" tanpa sebarang jaminan, sama ada nyata atau tersirat. Kami tidak menjamin bahawa laman web akan tersedia pada setiap masa atau bahawa ia akan bebas dari ralat atau virus.

Kami tidak membuat sebarang jaminan tentang ketepatan, kelengkapan, atau kebolehpercayaan sebarang maklumat di laman web ini.

Had Liabiliti

Setakat yang dibenarkan oleh undang-undang, kami tidak akan bertanggungjawab atas sebarang ganti rugi tidak langsung, sampingan, khas, berbangkit, atau punitif yang timbul daripada atau berkaitan dengan penggunaan anda terhadap laman web ini.

Ini termasuk, tetapi tidak terhad kepada, ganti rugi untuk kehilangan keuntungan, data, atau kerugian tidak ketara lain.

Pautan Luaran

Laman web kami mungkin mengandungi pautan ke laman web luaran yang tidak dikendalikan oleh kami. Kami tidak mempunyai kawalan ke atas kandungan dan amalan laman-laman ini dan tidak boleh menerima tanggungjawab atas dasar privasi atau kandungan mereka.

Perubahan kepada Terma

Kami berhak untuk mengubah suai Terma dan Syarat ini pada bila-bila masa. Perubahan akan berkuat kuasa serta-merta selepas disiarkan di halaman ini.

Penggunaan berterusan anda terhadap laman web selepas sebarang perubahan membentuk penerimaan anda terhadap terma baharu.

Undang-undang yang Mengawal

Terma dan Syarat ini dikawal oleh dan ditafsirkan mengikut undang-undang Sepanyol, dan anda tidak boleh ditarik balik untuk menyerahkan diri kepada bidang kuasa eksklusif mahkamah di lokasi tersebut.

Soalan?

Jika anda mempunyai sebarang soalan tentang Terma dan Syarat ini, sila hubungi kami.
