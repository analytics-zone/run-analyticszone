FAQ

Soalan Lazim

Bagaimanakah aplikasi analisis larian ini mendapatkan data saya?

Run Analytics senkron dengan Apple Health untuk mengimport latihan larian yang direkodkan oleh mana-mana peranti atau aplikasi yang serasi. Ini termasuk jam tangan pintar, penjejak kecergasan, dan entri manual. Aplikasi ini memproses data ini secara tempatan untuk mengira metrik prestasi larian termaju.

Apakah ujian kelajuan larian kritikal dan bagaimanakah cara saya melakukannya?

Kelajuan larian kritikal (CRS) ialah protokol saintifik menggunakan 2 larian usaha maksimum: 5K dan 3K dengan rehat 10-20 minit di antaranya. Aplikasi mengira ambang aerobik anda daripada masa-masa ini dan melaraskan semua zon latihan larian secara automatik. Ulang setiap 6-8 minggu untuk menjejaki kemajuan.

Adakah data larian saya dimuat naik ke awan?

Tidak. Run Analytics memproses semua data larian secara tempatan pada iPhone anda. Tiada pelayan luaran, tiada akaun awan, tiada pemindahan data. Anda mengawal eksport: jana fail JSON, CSV, HTML, atau PDF dengan metrik prestasi larian anda dan kongsikannya mengikut kemahuan anda.

Bolehkah saya menggunakan aplikasi analisis larian ini untuk larian trail?

Ya. Run Analytics berfungsi dengan mana-mana latihan larian dalam Apple Health, termasuk larian trail. Aplikasi ini menyesuaikan metrik kecekapan larian kepada data yang tersedia sama ada anda berada di trek atau trail, menyediakan analisis prestasi yang relevan untuk setiap persekitaran.

Apakah perbezaan antara pelan bulanan dan tahunan?

Kedua-dua pelan menawarkan ciri yang serupa: semua metrik prestasi larian, zon latihan tanpa had, perbandingan temporal, pelbagai format eksport, dan kemas kini percuma. Satu-satunya perbezaan ialah harga: tahunan menjimatkan 18% (bersamaan dengan ‚Ç¨3.25/bulan berbanding ‚Ç¨3.99/bulan).

Bolehkah saya membatalkan langganan saya pada bila-bila masa?

Ya. Langganan diuruskan melalui App Store, jadi anda boleh membatalkan pada bila-bila masa daripada Tetapan ‚Üí [Nama Anda] ‚Üí Langganan. Jika anda membatalkan, anda akan mengekalkan akses sehingga akhir tempoh pengebilan semasa anda.
