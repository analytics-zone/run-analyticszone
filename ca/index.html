FAQ

Preguntes Freqüents

Com obté les meves dades aquesta app d'analítica de running?

Run Analytics sincronitza amb Apple Health per importar entrenaments de running enregistrats per qualsevol dispositiu o app compatible. Això inclou rellotges intel·ligents, monitors d'activitat i entrades manuals. L'app processa aquestes dades localment per calcular mètriques avançades de rendiment de running.

Què és el test de velocitat crítica de carrera i com el faig?

La velocitat crítica de carrera (CRS) és un protocol científic utilitzant 2 curses d'esforç màxim: 5K i 3K amb 10-20 minuts de descans entre elles. L'app calcula el teu llindar aeròbic a partir d'aquests temps i ajusta automàticament totes les zones d'entrenament. Repeteix cada 6-8 setmanes per seguir el progrés.

Les meves dades de running es pugen al núvol?

No. Run Analytics processa totes les dades de running localment al teu iPhone. No hi ha servidors externs, ni comptes al núvol, ni transferències de dades. Tu controles les exportacions: genera fitxers JSON, CSV, HTML o PDF amb les teves mètriques de rendiment i comparteix-los com vulguis.

Puc utilitzar aquesta app d'analítica de running per a trail running?

Sí. Run Analytics funciona amb qualsevol entrenament de running a Apple Health, incloent trail running. L'app adapta les mètriques d'eficiència a les dades disponibles ja siguis a pista o trail, proporcionant anàlisi de rendiment rellevant per a cada entorn.

Quina diferència hi ha entre els plans mensual i anual?

Ambdós plans ofereixen funcions idèntiques: totes les mètriques de rendiment, zones d'entrenament il·limitades, comparacions temporals, múltiples exportacions i actualitzacions gratuïtes. L'única diferència és el preu: l'anual estalvia un 18% (equivalent a 3,25€/mes vs 3,99€/mes).

Puc cancel·lar la subscripció en qualsevol moment?

Sí. Les subscripcions es gestionen a través de l'App Store, així que pots cancel·lar quan vulguis des de Configuració → [El teu nom] → Subscripcions. Si cancel·les, mantindràs l'accés fins al final del teu període de facturació actual.
