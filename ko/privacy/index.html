개인정보 처리방침

최종 업데이트: 2025년 10월 27일

귀하의 개인정보는 보호됩니다

Run Analytics는 귀하의 개인 데이터를 수집, 저장 또는 전송하지 않습니다. 모든 데이터 처리는 귀하의 기기에서 로컬로 이루어집니다.

개요

Run Analytics는 귀하의 개인정보 보호를 최우선으로 설계되었습니다. 이 개인정보 처리방침은 귀하의 개인 정보 및 데이터를 보호하기 위한 당사의 약속을 설명합니다.

요약하자면: 당사는 필요하지 않기 때문에 귀하의 데이터를 수집하지 않습니다. 앱에서 수행하는 모든 작업은 귀하의 기기에 유지됩니다.

수집하지 않는 정보

Run Analytics는 다음을 수집, 전송 또는 저장하지 않습니다:

  • 개인 정보 (이름, 이메일, 전화번호 등)
  • 러닝 활동 데이터 (거리, 페이스, 경로, 시간 등)
  • 건강 데이터 (심박수, 칼로리 등)
  • 위치 데이터 또는 GPS 좌표
  • 기기 정보 또는 식별자
  • 사용 통계 또는 분석
  • 충돌 보고서 또는 진단
  • 기타 개인 또는 민감한 정보

앱 작동 방식

Run Analytics의 모든 계산 및 데이터 처리는 전적으로 귀하의 기기에서 이루어집니다:

  • 귀하의 러닝 데이터는 기기의 로컬 저장소에만 저장됩니다
  • 모든 계산 (페이스, 스플릿, 구간 등)은 로컬에서 수행됩니다
  • 데이터는 당사 서버나 제3자 서비스로 전송되지 않습니다
  • 앱이 작동하는 데 인터넷 연결이 필요하지 않습니다

데이터 저장

앱에서 입력하거나 생성한 모든 데이터는 iOS 로컬 저장 메커니즘을 사용하여 귀하의 기기에만 독점적으로 저장됩니다. 이 데이터는 귀하의 완전한 통제 하에 있습니다:

  • 귀하의 데이터는 항상 귀하의 기기에 유지됩니다
  • 당사는 귀하의 데이터에 액세스할 수 없습니다
  • 앱을 삭제하면 모든 데이터가 기기에서 영구적으로 제거됩니다
  • iOS 앱 백업을 명시적으로 활성화하지 않는 한 데이터는 iCloud 또는 클라우드 서비스에 동기화되지 않습니다

제3자 서비스

Run Analytics는 다음을 포함한 어떠한 제3자 서비스도 사용하지 않습니다:

  • 분석 서비스 없음 (Google Analytics, Firebase 등)
  • 광고 네트워크 없음
  • 충돌 보고 서비스 없음
  • 소셜 미디어 통합 없음
  • 결제 프로세서 없음 (앱은 무료입니다)

권한

앱은 귀하가 특정 기능을 사용하기로 선택한 경우에만 다음 iOS 권한을 요청할 수 있습니다:

  • HealthKit/건강 앱: 운동 데이터를 가져오기로 선택한 경우에만 해당됩니다. 모든 데이터는 기기에 유지됩니다.
  • 알림: 운동 알림을 활성화한 경우에만 해당됩니다. 데이터는 기기 외부로 전송되지 않습니다.

모든 권한은 선택 사항이며 iOS 설정을 통해 언제든지 철회할 수 있습니다.

아동 개인정보 보호

당사는 어떠한 데이터도 수집하지 않으므로 Run Analytics는 13세 미만의 어린이를 포함한 모든 연령대의 사용자에게 안전합니다. 그러나 모든 피트니스 활동에 대해 부모의 지도를 권장합니다.

이 개인정보 처리방침의 변경

당사는 앱 또는 법적 요구 사항의 변경 사항을 반영하기 위해 이 개인정보 처리방침을 수시로 업데이트할 수 있습니다. 변경 사항은 업데이트된 "최종 업데이트" 날짜와 함께 이 페이지에 게시됩니다.

중요: 귀하의 데이터를 수집하지 않겠다는 당사의 핵심 약속은 절대 변경되지 않습니다.

문의하기

이 개인정보 처리방침이나 앱에 대해 질문이 있는 경우 문의 페이지를 통해 문의해 주십시오.

요약

Run Analytics는 개인정보 보호를 최우선으로 하는 앱입니다. 당사는 귀하의 데이터가 귀하의 소유이며 귀하와 함께 있어야 한다고 믿습니다.

  • ❌ 데이터 수집 없음
  • ❌ 추적 또는 분석 없음
  • ❌ 제3자 서비스 없음
  • ✅ 100% 로컬 처리
  • ✅ 데이터에 대한 완전한 통제
