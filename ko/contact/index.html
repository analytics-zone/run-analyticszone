Run Analytics 문의하기

러닝 분석에 대한 질문, CRS 테스트 지원, 버그 제보, 기능 제안 등 어떤 것이든 환영합니다.

지원 요청 및 피드백

Run Analytics 팀은 러너와 트라이애슬리트가 훈련 데이터를 최대한 활용하도록 돕습니다. 영업일 기준 24~48시간 내에 답변을 드립니다.

무엇을 도와드릴까요

기술 지원

  • CRS 테스트 문제 해결
  • rTSS 계산 관련 문의
  • 훈련 존 설정 지원
  • 데이터 가져오기/내보내기 문제
  • 앱 기능 관련 질문

기능 요청

  • 새 지표 제안
  • 연동/통합 요청
  • 훈련 계획 기능
  • 데이터 시각화 아이디어
  • 워크플로 개선

버그 제보

  • 앱 크래시 또는 오류
  • 계산 정확도 문제
  • 화면 표시 문제
  • 동기화 이슈
  • 성능 관련 우려

일반 문의

  • 구독 관련 문의
  • 훈련 조언
  • 리서치 협업
  • 파트너십 제안
  • 미디어 문의

문의 전에 확인하세요

자주 묻는 질문은 아래 가이드에서 빠르게 확인할 수 있습니다.

  • 시작 가이드 - 온보딩 및 CRS 테스트 튜토리얼
  • CRS 가이드 - 임계 러닝 속도 이해하기
  • TSS 가이드 - Training Stress Score 설명
  • 훈련 존 - 5존 시스템 정리
  • 연구 - 피어 리뷰 기반 자료

이 자료에서 더 빨리 답을 찾을 수 있습니다.
