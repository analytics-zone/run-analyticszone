전문적인 러닝 분석

더 스마트하게 훈련하고, 더 빠르게 달리세요

과학적인 러닝 퍼포먼스 지표, 맞춤형 트레이닝 존, 그리고 포괄적인 퍼포먼스 추적 기능을 갖춘 프라이버시 우선 iOS 앱입니다. 모든 데이터는 iPhone에서 로컬로 처리되어 완벽한 개인정보 보호를 보장합니다.

기능

고급 러닝 퍼포먼스 지표

모든 수준의 러너를 위해 설계된 전문적인 러닝 분석

과학적인 러닝 지표

임계 러닝 속도 (CRS) 가 귀하의 유산소 작업 임계값을 결정하여, 트레이닝 스트레스 점수 (TSS) 계산 및 과학적인 스포츠 연구에 기반한 퍼포먼스 추적 CTL/ATL/TSB 를 가능하게 합니다.

개인별 맞춤 트레이닝 존

러닝 트레이닝 존 임계 속도에 따라 보정된 6개의 맞춤형 러닝 트레이닝 존. 회복, 유산소 기초 체력 강화, 임계값 훈련, 또는 VO₂max 향상을 위해 각 훈련을 최적화하세요.

퍼포먼스 비교

주간, 월간, 연간 단위의 기간 비교. 모든 퍼포먼스 지표에 대해 자동 트렌드 분석 및 변화율 확인이 가능합니다.

완벽한 프라이버시 보호

모든 러닝 데이터는 iOS 기기 내에서 로컬로 처리됩니다. 서버 저장, 클라우드 백업, 추적 등이 전혀 없습니다. 귀하의 러닝 분석 데이터는 오직 귀하만이 제어합니다.

어디로든 데이터 내보내기

워크아웃과 퍼포먼스 지표를 JSON, CSV, HTML, PDF 형식으로 내보낼 수 있습니다. 코치, 스프레드시트 및 타 플랫폼과의 호환성을 제공합니다.

즉각적인 응답 속도

로컬 우선(local-first) 아키텍처를 통해 앱이 0.35초 이내에 실행됩니다. 동기화를 기다릴 필요 없이 즉시 러닝 분석 데이터를 확인하세요.

복잡함 없이 누리는 전문적인 러닝 퍼포먼스 지표

CRS 테스트 프로토콜

임계 러닝 속도를 결정하기 위한 1200m/3600m 테스트 프로토콜 제공. 6~8주마다 테스트하여 발전 정도를 확인하고 트레이닝 존을 자동으로 갱신하세요.

iOS 네이티브 러닝 앱

SwiftUI로 개발되어 부드러운 성능과 완벽한 iOS 통합을 제공합니다. Apple 건강 앱 동기화, 위젯 지원, 그리고 친숙한 Apple 디자인을 경험하세요.

연구 기반의 지표

모든 지표는 동료 검토를 거친 과학적 연구에 기반합니다. Wakayoshi 등의 CRS, IF² 공식을 적용한 러닝용 TSS, 입증된 CTL/ATL 모델을 사용합니다.

코치용 리포트 제공

코치를 위한 상세 리포트를 내보낼 수 있습니다. 이메일로 HTML 요약을 공유하거나, 엑셀 분석용 CSV, 기록 보관용 PDF 파일을 생성하세요.

장소에 구애받지 않는 분석

트랙, 트레일, 스프린트, 장거리 달리기 등 모든 환경에서 작동합니다. Run Analytics는 모든 종류의 러닝에 효율 지표를 적용하고 훈련 특성을 자동 감지합니다.

지속적인 발전

사용자 피드백을 바탕으로 정기적으로 기능을 업데이트합니다. 최근 연간 비교 기능, 개인 기록(PR) 추적, 강화된 데이터 내보내기 옵션 등이 추가되었습니다.

FAQ

앱은 내 러닝 데이터를 어떻게 가져오나요?

Run Analytics는 Apple 건강 앱과 동기화하여 호환되는 모든 기기나 앱으로 기록된 러닝 데이터를 가져옵니다. 여기에는 스마트워치, 피트니스 트래커, 수동 입력 데이터가 포함됩니다. 앱은 이 데이터를 로컬에서 처리하여 고급 퍼포먼스 지표를 계산합니다.

임계 러닝 속도 테스트가 무엇이며 어떻게 하나요?

임계 러닝 속도(CRS)는 5K와 3K를 전력으로 달리고 그 사이에 10~20분 휴식을 취하는 과학적 프로토콜입니다. 앱은 해당 기록을 통해 유산소 임계값을 계산하고 모든 트레이닝 존을 자동으로 설정합니다. 6~8주마다 반복하여 기량을 체크하세요.

내 데이터가 클라우드에 업로드되나요?

아니요. Run Analytics는 모든 러닝 데이터를 iPhone 내에서 로컬로 처리합니다. 외부 서버, 클라우드 계정, 데이터 전송이 전혀 없습니다. 데이터 내보내기는 귀하가 직접 제어하며, JSON, CSV, HTML, PDF 파일로 생성하여 원하는 곳에 공유할 수 있습니다.

트레일 러닝에도 이 분석 앱을 사용할 수 있나요?

네. Run Analytics는 Apple 건강 앱에 기록된 모든 러닝 데이터(트레일 러닝 포함)와 작동합니다. 앱은 트랙이든 산길이든 사용 가능한 데이터에 맞춰 효율 지표를 조정하여 환경에 맞는 분석 결과를 제공합니다.

월간 요금제와 연간 요금제의 차이점은 무엇인가요?

두 요금제 모두 기능은 동일합니다(모든 지표, 무제한 존, 기간 비교, 다양한 데이터 내보내기 등). 차이점은 가격뿐이며, 연간 요금제 선택 시 약 17%를 절약할 수 있습니다(월 평균 약 ₩4,580 vs ₩5,500).

언제든지 구독을 취소할 수 있나요?

네. 구독은 App Store를 통해 관리되므로 설정 → [사용자 이름] → 구독 메뉴에서 언제든지 취소할 수 있습니다. 취소하더라도 현재 결제 기간이 끝날 때까지는 모든 기능을 계속 사용할 수 있습니다.

러닝 실력을 한 단계 업그레이드할 준비가 되셨나요?

과학적인 분석을 통해 기량을 향상시키고 있는 수천 명의 러너들과 함께하세요. 프라이버시 중심의 iOS 러닝 분석 앱을 7일 무료 체험으로 지금 시작해 보세요.
