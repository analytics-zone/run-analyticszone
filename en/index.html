FAQ

Frequently Asked Questions

How does this running analytics app get my data?

RunAnalytics syncs with Apple Health to import running workouts recorded by any compatible device or app. This includes smart watches, fitness trackers, and manual entries. The app processes this data locally to calculate advanced running performance metrics.

What is the critical running speed test and how do I perform it?

Critical running speed (CRS) is a scientific protocol using 2 maximum effort runs: 5K and 3K with 10-20 minutes rest between. The app calculates your aerobic threshold from these times and automatically adjusts all running training zones. Repeat every 6-8 weeks to track progress.

Is my running data uploaded to the cloud?

No. RunAnalytics processes all running data locally on your iPhone. There are no external servers, no cloud accounts, no data transfers. You control exports: generate JSON, CSV, HTML, or PDF files with your running performance metrics and share them however you want.

Can I use this running analytics app for trail running?

Yes. RunAnalytics works with any running workout in Apple Health, including trail running. The app adapts running efficiency metrics to available data whether you're on track or trail, providing relevant performance analysis for each environment.
