Running Efficiency: Running Efficiency

Your Stride Economy Score - Lower is Better

What is Running Efficiency?

Running Efficiency (Run + Golf) is a composite efficiency metric that combines stride count and time into a single number. Like golf, the goal is to minimize your score.

Formula

Running Efficiency = Kilometer Time (seconds) + Stride Count

Example: If you run 25m in 20 seconds with 15 strides:

Running Efficiency = 20 + 15 = 35

Normalized Running Efficiency for Track Comparison

To compare scores across different track lengths:

Running Efficiency₂₅ = (Time × 25/Track Length) + (Strides × 25/Track Length)

⚠️ Individual Variation

Running Efficiency is influenced by height and arm span. Taller runners naturally take fewer strides. Use Running Efficiency to track your own progress rather than comparing to others.

Interpreting Running Efficiency Patterns

📉 Decreasing Running Efficiency = Improving Efficiency

Your technique is getting better, or you're becoming more economical at a given pace. This is the goal over weeks and months of training.

Example: Running Efficiency drops from 48 → 45 → 42 over 8 weeks of focused technique work.

📈 Increasing Running Efficiency = Declining Efficiency

Fatigue is setting in, technique is breaking down, or you're running faster than your efficiency allows.

Example: Running Efficiency rises from 42 → 48 during final 200m of a 1000m set, indicating fatigue.

📊 Different Combinations at Same Running Efficiency

A Running Efficiency of 45 could result from multiple stride/time combinations:

  • 20 seconds + 25 strides = High frequency, shorter strides
  • 25 seconds + 20 strides = Lower frequency, longer strides

Always analyze the components (stride count AND time) to understand your running strategy.

🎯 Running Efficiency Training Applications

  • Technique Sessions: Aim to reduce Running Efficiency through better catch, streamline, and body position
  • Fatigue Monitoring: Rising Running Efficiency indicates technique breakdown—time for a break
  • Pace-Efficiency Balance: Find the fastest pace you can hold without Running Efficiency spiking
  • Drill Effectiveness: Track Running Efficiency before/after drill sets to measure technique transfer

Measurement Best Practices

📏 Stride Counting

  • Count each hand entry (both arms combined)
  • Start counting from first stride after push-off
  • Count up to wall touch
  • Maintain consistent push-off distance (~5m from flags)

⏱️ Timing

  • Measure from first stride to wall touch
  • Use consistent push-off intensity across segments
  • Technology (Garmin, Apple Watch, FORM) auto-calculates
  • Manual timing: Use pace clock or stopwatch

🔄 Consistency

  • Measure Running Efficiency at similar paces for comparison
  • Track during main sets, not warm-up/cool-down
  • Note which stride type (freestyle, back, etc.)
  • Compare same track length (25m vs 25m, not 25m vs 50m)

Running Efficiency Limitations

🚫 Cannot Compare Between Athletes

Height, arm length, and flexibility create natural stride count differences. A 6'2" runner will have a lower Running Efficiency than a 5'6" runner at the same fitness level.

Solution: Use Running Efficiency for personal progress tracking only.

🚫 Composite Score Hides Details

Running Efficiency combines two variables. You can improve one while worsening the other and still have the same score.

Solution: Always examine stride count AND time separately.

🚫 Not Pace-Normalized

Running Efficiency naturally increases as you run faster (more strides, less time, but higher total). This isn't inefficiency—it's physics.

Solution: Track Running Efficiency at specific target paces (e.g., "Running Efficiency at CRS pace" vs "Running Efficiency at easy pace").

🔬 The Science Behind Running Economy

Research by Costill et al. (1985) established that running economy (energy cost per unit distance) is more important than VO₂max for middle-distance performance.

Running Efficiency serves as a proxy for economy—lower Running Efficiency typically correlates with lower energy expenditure at a given pace, allowing you to run faster or longer with the same effort.

Running Efficiency Training Drills

🎯 Running Efficiency Reduction Set

8 × 50m (30 seconds rest)

  1. 50 #1-2: Run at comfortable pace, record baseline Running Efficiency
  2. 50 #3-4: Reduce stride count by 2, maintain same time → Focus on length per stride
  3. 50 #5-6: Increase stride rate slightly, keep stride count same → Focus on turnover
  4. 50 #7-8: Find optimal balance—aim for lowest Running Efficiency

Goal: Discover your most efficient stride count/rate combination.

⚡ Running Efficiency Stability Test

10 × 100m @ CRS Pace (20 seconds rest)

Record Running Efficiency for each 100m. Analyze:

  • Which 100m had lowest Running Efficiency? (You were most efficient)
  • Where did Running Efficiency spike? (Technique breakdown or fatigue)
  • How much did Running Efficiency drift from first to last 100m?

Goal: Maintain Running Efficiency ±2 points across all reps. Consistency indicates strong technique under fatigue.

Frequently Asked Questions

What is Running Efficiency?

Running Efficiency (Run + Golf) is a combined efficiency metric that adds your stride count to your time for a length. Like in golf, the goal is to minimize your score. For example, 20 seconds + 15 strides = Running Efficiency 35.

How do I calculate my Running Efficiency?

Count every stride (each hand entry) for a length and add your time in seconds. Running Efficiency = Time (seconds) + Stride Count. Some watches calculate this automatically.

What is a good Running Efficiency score?

For 25m freestyle: Elite runners score 30-35, competitive runners 35-45, fitness runners 45-60, beginners 60+. Your height and arm span affect stride count, so focus on improving your own score over time rather than comparing with others.

Can I compare my Running Efficiency with other runners?

No. Running Efficiency is highly individual because taller runners naturally take fewer strides. Use Running Efficiency to track your own progress, not to compare with others. A tall runner with poor technique might have the same Running Efficiency as a shorter runner with excellent technique.

Should Running Efficiency go up or down as I run faster?

Running Efficiency naturally increases slightly as you run faster due to physics - you need more strides per second. Focus on Running Efficiency at specific, consistent paces. Track 'Running Efficiency at easy pace' vs 'Running Efficiency at threshold pace' separately.

Why is my Running Efficiency getting worse during a set?

Increasing Running Efficiency during a set indicates fatigue causing technique breakdown. This is normal and shows where your technique falls apart under stress. Use this information to identify technical weaknesses to work on.

Can I use Running Efficiency for backstride, breaststride, or butterfly?

Yes, but stride-specific benchmarks differ. Backstride is typically 5-10 points higher than freestyle. Breaststride has a wide range due to glide technique. Butterfly is similar to freestyle for skilled runners. Track each stride separately.

How can I improve my Running Efficiency?

Focus on technique: longer strides (better catch and pull-through), improved streamline (off walls and during stride), better body position (reduce drag), and consistent rotation. Drills and video analysis help identify specific areas for improvement. Learn more in our Stride Mechanics guide.
