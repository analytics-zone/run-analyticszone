การวิเคราะห์การวิ่งระดับมืออาชีพที่ออกแบบมาสำหรับนักวิ่งทุกระดับ

ตัวชี้วัดการวิ่งทางวิทยาศาสตร์

ความเร็วในการวิ่งวิกฤต (CRS) กำหนดเกณฑ์แอโรบิกของคุณ เปิดใช้งานการคำนวณ คะแนนความเครียดในการฝึกซ้อม (TSS) และการติดตาม CTL/ATL/TSB ตามหลักวิทยาศาสตร์การกีฬาที่ได้รับการพิสูจน์แล้ว

โซนการฝึกซ้อมส่วนบุคคล

โซนการฝึกซ้อมการวิ่งแบบส่วนบุคคลตาม 5 โซน หรือ 7 โซน ที่ปรับตามความเร็ววิกฤตของคุณ เพิ่มประสิทธิภาพทุกการออกกำลังกาย ตั้งแต่การฟื้นตัว การสร้างพื้นฐานแอโรบิก ไปจนถึงการฝึก VO₂max

การเปรียบเทียบประสิทธิภาพ

เปรียบเทียบข้อมูลรายสัปดาห์ รายเดือน และรายปี พร้อมการตรวจจับแนวโน้มอัตโนมัติและการเปลี่ยนแปลงเป็นเปอร์เซ็นต์สำหรับทุกตัวชี้วัดประสิทธิภาพการวิ่ง

ความเป็นส่วนตัวที่สมบูรณ์

ข้อมูลการวิ่งทั้งหมดประมวลผลภายในอุปกรณ์ iOS ของคุณ 100% ไม่มีเซิร์ฟเวอร์ ไม่มีการเก็บข้อมูลบนคลาวด์ ไม่มีการติดตาม คุณเป็นเจ้าของและควบคุมข้อมูลการวิ่งของคุณอย่างสมบูรณ์

ส่งออกข้อมูลได้ทุกที่

ส่งออกข้อมูลการออกกำลังกายและตัวชี้วัดการวิ่งในรูปแบบ JSON, CSV, HTML หรือ PDF เพื่อทำงานร่วมกับโค้ช สเปรดชีต หรือแพลตฟอร์มการฝึกซ้อมอื่นๆ

ทำไมต้อง Run Analytics

การวิเคราะห์การวิ่งที่เน้นความเป็นส่วนตัว สำหรับนักกีฬาที่จริงจัง

ตัวชี้วัดระดับมืออาชีพที่ใช้งานง่าย ไม่ซับซ้อน

โปรโตคอลการทดสอบ CRS

มาพร้อมโปรโตคอลการทดสอบ 1,200 เมตร และ 3,600 เมตร เพื่อกำหนดความเร็ววิกฤตของคุณ ทดสอบซ้ำทุก 6-8 สัปดาห์เพื่อติดตามความก้าวหน้าและปรับโซนการฝึกซ้อมโดยอัตโนมัติ

แอป iOS แบบ Native

พัฒนาด้วย SwiftUI เพื่อประสิทธิภาพที่ราบรื่นและการผสมผสานกับ iOS อย่างสมบูรณ์แบบ ซิงค์กับแอป Apple Health เพื่อการวิเคราะห์ที่รวดเร็ว พร้อมรองรับวิดเจ็ตและดีไซน์ที่คุ้นเคย

ตัวชี้วัดจากงานวิจัย

ทุกตัวชี้วัดอ้างอิงจากงานวิจัยทางวิทยาศาสตร์ที่ผ่านการรับรอง CRS จาก Wakayoshi และคณะ คะแนน TSS ที่ปรับให้เหมาะกับการวิ่ง และโมเดล CTL/ATL ที่ได้รับการพิสูจน์แล้ว

รายงานที่ส่งต่อให้โค้ชได้ง่าย

ส่งออกรายงานละเอียดเพื่อแชร์กับโค้ช ทั้งไฟล์ HTML สรุปทางอีเมล, CSV สำหรับวิเคราะห์ในสเปรดชีต หรือ PDF สำหรับบันทึกประวัติการฝึกซ้อม

ใช้งานได้ทุกที่

ไม่ว่าจะวิ่งบนลู่วิ่งหรือทางเทรล ในเมืองหรือในป่า Run Analytics จะปรับตัวชี้วัดประสิทธิภาพให้เหมาะกับการวิ่งทุกประเภทและตรวจจับลักษณะการออกกำลังกายโดยอัตโนมัติ

พัฒนาอย่างไม่หยุดนิ่ง

อัปเดตฟีเจอร์ใหม่ๆ สม่ำเสมอตามคำแนะนำของผู้ใช้งาน ล่าสุดเราได้เพิ่มการเปรียบเทียบรายปี การติดตามสถิติส่วนบุคคล และปรับปรุงตัวเลือกการส่งออกข้อมูลให้ดียิ่งขึ้น

คำถามที่พบบ่อย (FAQs)

แอปดึงข้อมูลการวิ่งของฉันจากไหน?

Run Analytics ซิงค์กับแอป Apple Health เพื่อนำเข้าการออกกำลังกายที่บันทึกโดยอุปกรณ์ที่รองรับ (เช่น Apple Watch, Garmin) หรือแอปอื่นๆ ข้อมูลจะถูกประมวลผลภายในเครื่องเพื่อคำนวณตัวชี้วัดขั้นสูง

การทดสอบความเร็ววิกฤต (CRS) คืออะไร และต้องทำอย่างไร?

CRS คือโปรโตคอลทางวิทยาศาสตร์ที่ใช้การวิ่งเต็มกำลัง 2 ช่วง (เช่น 3,600 ม. และ 1,200 ม.) แอปจะคำนวณเกณฑ์แอโรบิกจากช่วงเวลาเหล่านี้และปรับโซนการฝึกซ้อมให้คุณโดยอัตโนมัติ

ข้อมูลการวิ่งของฉันถูกอัปโหลดขึ้นคลาวด์หรือไม่?

ไม่ ข้อมูลการวิ่งทั้งหมดจะถูกประมวลผลภายใน iPhone ของคุณเท่านั้น ไม่มีเซิร์ฟเวอร์ภายนอกและไม่มีการเก็บข้อมูลบนคลาวด์ คุณเป็นผู้ควบคุมข้อมูล 100%

แอปนี้รองรับการวิ่งเทรล (Trail Running) หรือไม่?

รองรับแน่นอน Run Analytics ใช้งานได้กับการออกกำลังกายทุกประเภทที่บันทึกใน Apple Health รวมถึงการวิ่งเทรลด้วย โดยจะวิเคราะห์ประสิทธิภาพตามข้อมูลที่มีให้อย่างแม่นยำ

สมัครสมาชิกรายเดือนกับรายปีต่างกันอย่างไร?

ฟีเจอร์เหมือนกันทุกประการ ทั้งตัวชี้วัดขั้นสูง โซนการฝึกซ้อม และการส่งออกข้อมูล ต่างกันเพียงราคา โดยรายปีจะประหยัดกว่าถึง 18% (เฉลี่ย €3.25 ต่อเดือน)

สามารถยกเลิกการสมาชิกได้ทุกเมื่อใช่ไหม?

ได้แน่นอน คุณสามารถจัดการการสมัครสมาชิกผ่าน App Store ในเมนูการตั้งค่าของ Apple ID คุณจะยังใช้งานฟีเจอร์ Pro ได้จนกว่าจะสิ้นสุดรอบบิลปัจจุบัน

พร้อมที่จะยกระดับการวิ่งของคุณหรือยัง?

ร่วมกับนักวิ่งมากมายที่ใช้การวิเคราะห์การวิ่งทางวิทยาศาสตร์เพื่อพัฒนาประสิทธิภาพ เริ่มทดลองใช้ฟรี 7 วันวันนี้ กับแอป iOS ที่ให้ความสำคัญกับความเป็นส่วนตัวของคุณ
