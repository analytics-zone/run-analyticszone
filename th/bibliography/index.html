บรรณานุกรมที่สมบูรณ์

ข้อมูลอ้างอิงทางวิทยาศาสตร์ที่สนับสนุน Run Analytics

วรรณกรรมวิทยาศาสตร์อ้างอิง

หน่วยวัดและสูตรทั้งหมดใน Run Analytics ได้รับการสนับสนุนโดยงานวิจัยที่ผ่านการตรวจสอบโดยผู้ทรงคุณวุฒิซึ่งเผยแพร่ใน วารสารวิทยาศาสตร์การกีฬา สรีรวิทยา และชีวกลศาสตร์ชั้นนำ

ครอบคลุมวารสาร

การอ้างอิงครอบคลุมสิ่งพิมพ์รวมถึง:

  • วารสารสรีรวิทยาประยุกต์แห่งยุโรป
  • วารสารเวชศาสตร์การกีฬานานาชาติ
  • วารสารวิทยาศาสตร์การกีฬา
  • เวชศาสตร์การกีฬา
  • วารสารการวิจัยความแข็งแกร่งและการปรับสภาพ
  • เซ็นเซอร์ (MDPI)
  • วิทยาศาสตร์การเคลื่อนไหวของมนุษย์
  • ธรรมชาติ
  • พรมแดนทางสรีรวิทยา
