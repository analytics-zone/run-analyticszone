  • តំបន់បណ្តុះបណ្តាលត្រឹមត្រូវ៖ការធ្វើតេស្តបង្កើតតំបន់អាំងតង់ស៊ីតេច្បាស់លាស់ដោយផ្អែកលើសរីរវិទ្យាពិតប្រាកដរបស់អ្នក មិនមែនរូបមន្តទូទៅ ឬភាគរយនៃអត្រាបេះដូងអតិបរមា

  • ប្រព័ន្ធព្រមានជាមុន៖ការថយចុះលទ្ធផលតេស្តអាចបង្ហាញពីការហ្វឹកហ្វឺនហួសកម្រិត ការងើបឡើងវិញមិនគ្រប់គ្រាន់ ឬត្រូវការការកែតម្រូវការបណ្តុះបណ្តាល

គណនា CRS៖

CRS (m/min) = (D7 - D3) / (7 - 3)

កន្លែងណា៖
D7 = ចម្ងាយគ្របដណ្តប់ក្នុងការសាកល្បង 7 នាទី (ម៉ែត្រ)
D3 = ចម្ងាយគ្របដណ្តប់ក្នុងការសាកល្បង 3 នាទី (ម៉ែត្រ)

ឧទាហរណ៍ការគណនា

សវនាការ ចម្ងាយ ការគណនា
សាកល្បង 3 នាទី 900 ម៉ែត្រ D3 = 900 ម
សាកល្បង 7 នាទី 1,980 ម៉ែត្រ D7 = 1,980 ម
CRS លទ្ធផល (1,980 - 900) / (7 - 3) = 270 m/min = 3:42/km = 5:57/mile

ប្រើ CRS របស់អ្នកពីការធ្វើតេស្តដើម្បីគណនាrTSS (ការរត់ការហ្វឹកហាត់ពិន្ទុ)សម្រាប់រាល់ការហាត់ប្រាណ។ វាកំណត់បរិមាណនៃការហ្វឹកហាត់ និងជួយគ្រប់គ្រងភាពអស់កម្លាំង៖
