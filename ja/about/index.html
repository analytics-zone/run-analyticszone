Run Analyticsについて

ランナーによって、ランナーのために構築された科学的根拠に基づくランニングパフォーマンス追跡

私たちのミッション

Run Analyticsは、すべてのランナーにプロフェッショナルグレードのパフォーマンス追跡をもたらします。クリティカルランスピード（CRS）、トレーニングストレススコア（TSS）、パフォーマンス管理チャートなどの高度な指標が、高価なプラットフォームに閉じ込められたり、複雑なコーチングソフトウェアを必要としたりすべきではないと私たちは信じています。

開発者紹介

Albert Arnó

創設者・リード開発者

Albertは、15年以上のマスターズランニング経験を持つソフトウェア開発者であり、元競技ランナーです。特定のハードウェアやクラウドプラットフォームに縛られない、手頃な価格で科学的根拠に基づくランニング分析ツールを見つけるのに苦労した後、Run Analyticsを作成しました。

Run Analyticsが作られた理由：

「何年もの間、様々な追跡プラットフォームを使用した後、3つの繰り返し発生する問題にフラストレーションを感じていました：高価なサブスクリプション、特定のデバイスを必要とするベンダーロックイン、そして自分のデータに対するコントロールの欠如です。CRSとTSSを正しく計算し、Apple Health互換デバイスで動作し、データをプライベートに保つツールが欲しかったのです。見つからなかったので、作りました。」

「Run Analyticsは、初回の1:10/100m未満のCRSを目指してトレーニングしていた時に欲しかったアプリです。厳密なスポーツ科学（WakayoshiのCRS研究とCogganのTSS方法論に基づく）と現代的なiOSデザイン、完全なデータプライバシーを組み合わせています。」

私たちの原則

  • 科学第一： すべての指標は査読済み研究に基づいています。私たちは情報源を引用し、公式を示します。
  • プライバシー・バイ・デザイン： 100%ローカルデータ処理。サーバーなし、アカウントなし、トラッキングなし。あなたがデータを所有します。
  • プラットフォーム非依存： Apple Health互換デバイスで動作。ベンダーロックインなし。
  • 透明性： オープンな公式、明確な計算、正直な制限。ブラックボックスアルゴリズムなし。
  • アクセシビリティ： 高度な指標にスポーツ科学の学位は必要ありません。概念を明確に説明します。

科学的基盤

Run Analyticsは数十年にわたる査読済みスポーツ科学研究に基づいて構築されています：

クリティカルランスピード（CRS）

大阪大学のWakayoshi et al.（1992-1993）の研究に基づいています。CRSは疲労なしに持続可能な理論的最大ランニング速度を表し、乳酸閾値に対応します。

主要研究： Wakayoshi K, et al. "Determination and validity of critical velocity as an index of running performance." European Journal of Applied Physiology, 1992.

トレーニングストレススコア（TSS）

Dr. Andrew Cogganのサイクリング用TSS方法論をランニング用に適応。強度（CRSに対する相対値）と持続時間を組み合わせてトレーニング負荷を定量化。

主要研究： Coggan AR, Allen H. "Training and Racing with a Power Meter." VeloPress, 2010. Run AnalyticsがCRSを閾値として使用してランニング用に適応。

パフォーマンス管理チャート（PMC）

慢性トレーニング負荷（CTL）、急性トレーニング負荷（ATL）、トレーニングストレスバランス（TSB）指標。時間の経過とともにフィットネス、疲労、フォームを追跡。

実装： CTLは42日間の指数加重移動平均、ATLは7日間。TSB = CTL - ATL。

ランニング効率とストライド指標

時間とストライド数を組み合わせたランニング効率指標。世界中のエリートランナーとコーチが技術的改善を追跡するために使用。

標準指標： ランニング効率 = 時間 + ストライド。低いスコアはより良い効率を示します。ストライド距離（DPS）とストライドレート（SR）で補完。

開発とアップデート

Run Analyticsは、ユーザーフィードバックと最新のスポーツ科学研究に基づく定期的なアップデートで積極的に開発されています。アプリは以下で構築されています：

  • Swift & SwiftUI - 現代的なiOSネイティブ開発
  • HealthKit統合 - シームレスなApple Health同期
  • Core Data - 効率的なローカルデータストレージ
  • Swift Charts - 美しいインタラクティブなデータ可視化
  • サードパーティ分析なし - あなたの使用データはプライベートのまま

編集基準

Run Analyticsとこのウェブサイトのすべての指標と公式は、査読済みスポーツ科学研究に基づいています。私たちは原典を引用し、透明な計算を提供します。コンテンツは開発者（15年以上のランニング経験、コンピュータサイエンス修士）によって科学的正確性がレビューされています。
