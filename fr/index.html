FAQ

Questions fréquentes

Comment cette application d'analyse de cours obtient-elle mes données ?

Run Analytics se synchronise avec Apple Health pour importer les séances de cours enregistrées par n'importe quel appareil ou application compatible. Cela inclut les montres connectées, les trackers d'activité et les entrées manuelles. L'application traite ces données localisées pour calculer des métriques de performance de cours avancé.

Qu'est-ce que le test de vitesse critique de course et comment le réaliser ?

La vitesse critique de course (CRS) est un protocole scientifique utilisant 2 courses à effort maximal : 5K et 3K avec 10-20 minutes de repos entre elles. L'application calcule votre seuil aérobie à partir de ces temps et ajuste automatiquement toutes les zones d'entraînement. Répétez toutes les 6-8 semaines pour suivre les progrès.

Mes données de cours sont-elles téléchargées dans le cloud ?

Non. Run Analytics traite toutes les données de cours localement sur votre iPhone. Il n'y a pas de serveurs externes, pas de comptes cloud, pas de transferts de données. Vous contrôlez les exportations : généré des fichiers JSON, CSV, HTML ou PDF avec vos métriques de performance et partagez-les comme vous le souhaitez.

Puis-je utiliser cette application d'analyse de course pour le trail running ?

Oui. Run Analytics fonctionne avec n'importe quelle séance de course dans Apple Health, y compris le trail courir. L'application s'adapte les métriques d'efficacité aux données disponibles que vous soyez sur piste ou en trail, offrant une analyser les performances pertinente pour chaque environnement.

Quelle est la différence entre les forfaits mensuels et annuels ?

Les deux forfaits offrent des fonctionnalités identiques : toutes les métriques de performance, zones d'entraînement illimitées, comparaisons temporelles, multiples exports et mises à jour gratuites. La seule différence est le prix : l'annuel enregistreur 18% (équivalent à 3,25€/mois contre 3,99€/mois).

Puis-je annuler mon abonnement à tout moment ?

Oui. Les abonnements sont gérés via l'App Store, vous pouvez donc annuler à tout moment depuis Paramètres → [Votre nom] → Abonnements. Si vous annulez, vous conservez l'accès jusqu'à la fin de votre période de facturation en cours.
