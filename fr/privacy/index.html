politique de confidentialité

Dernière mise à jour :27 octobre 2025

Votre vie privée est protégée

Run Analytics ne collecte, ne stocke ni ne transmet aucune de vos données personnelles.Tout le traitement des données s'effectue localement sur votre appareil.

Aperçu

Run Analytics est conçu avec votre vie privée comme priorité absolue. Cette politique de confidentialité explique notre engagement à protéger vos informations et données personnelles.

En bref : nous ne collectons pas vos données parce que nous n'en avons pas besoin.Tout ce que vous faites dans l'application reste sur votre appareil.

Informations que nous ne collectons PAS

Run Analytics ne collecte, ne transmet ni ne stocke :

  • Informations personnelles (nom, email, numéro de téléphone, etc.)
  • Données d'activité de course à pied (distance, allure, itinéraires, temps, etc.)
  • Données de santé (fréquence cardiaque, calories, etc.)
  • Données de localisation ou coordonnées GPS
  • Informations ou identifiants sur l'appareil
  • Statistiques ou analyses d'utilisation
  • Rapports d'erreur ou diagnostics
  • Toute autre information personnelle ou sensible

Comment fonctionne l'application

Tous les calculs et traitements de données dans Run Analytics ont lieuentièrement sur votre appareil:

  • Vos données de course sont stockées uniquement dans le stockage local de votre appareil
  • Tous les calculs (allure, temps intermédiaires, zones, etc.) sont effectués localement
  • Aucune donnée n'est jamais envoyée à nos serveurs ou à des services tiers
  • Aucune connexion Internet n'est requise pour que l'application fonctionne

Stockage des données

Toutes les données que vous saisissez ou générez dans l'application sont stockées exclusivement sur votre appareil à l'aide des mécanismes de stockage local iOS. Ces données restent sous votre contrôle total :

  • Vos données restent sur votre appareil à tout moment
  • Nous ne pouvons pas accéder à vos données
  • IF vous supprimez l'application, toutes les données sont définitivement supprimées de votre appareil
  • Les données ne sont pas synchronisées avec iCloud ou tout autre service cloud, sauf si vous activez explicitement les sauvegardes de l'application iOS

Services tiers

Run Analytics n'utilise aucun service tiers, notamment :

  • Aucun service d'analyse (Google Analytics, Firebase, etc.)
  • Pas de réseaux publicitaires
  • Aucun service de rapport d'accident
  • Aucune intégration de médias sociaux
  • Aucun processeur de paiement (l'application est gratuite)

Autorisations

L'application peut demander les autorisations iOS suivantes uniquement IF si vous choisissez d'utiliser des fonctionnalités spécifiques :

  • HealthKit/Application Santé :Seulement IF vous choisissez d'importer les données d'entraînement. Toutes les données restent sur votre appareil.
  • Notifications :Seulement IF vous activez les rappels d'entraînement. Aucune donnée n'est envoyée en dehors de votre appareil.

Toutes les autorisations sont facultatives et vous pouvez les révoquer à tout moment via les paramètres iOS.

Confidentialité des enfants

Parce que nous ne collectons aucune donnée, Run Analytics est sans danger pour les utilisateurs de tous âges, y compris les enfants de moins de 13 ans. Cependant, nous recommandons la surveillance parentale pour toutes les activités de remise en forme.

Modifications de cette politique de confidentialité

Nous pouvons mettre à jour cette politique de confidentialité de temps à autre pour refléter les modifications apportées à l'application ou les exigences légales. Toute modification sera publiée sur cette page avec une date de « Dernière mise à jour » mise à jour.

Important:Nous ne modifierons jamais notre engagement fondamental de ne pas collecter vos données.

Contactez-nous

IF vous avez des questions sur cette politique de confidentialité ou sur l'application, veuillez nous contacter via notrepage de contact.

Résumé

Run Analytics est une application axée sur la confidentialité.Nous pensons que vos données vous appartiennent et doivent rester avec vous.

  • ❌ Aucune collecte de données
  • ❌ Pas de suivi ni d'analyse
  • ❌ Aucun service tiers
  • ✅ Transformation 100% locale
  • ✅ Contrôle total sur vos données
