Ano ang Critical Running Speed?

Ang Critical running speed (CRS) ay ang bilis na kaya mong takbuhin nang hindi agad napapagod. Kinakatawan nito ang iyong aerobic threshold pace. Ginagamit ang dalawang time trial (hal. 1200m at 3600m) upang malaman ang iyong personalized training zones para sa mas mahusay na pagsasanay.

üéØ Kahalagahang Pisyolohikal

Ang CRS ay malapit na kaugnay ng:

  • Lactate Threshold 2 (LT2) - Pangalawang ventilatory threshold
  • Maximal Lactate Steady State (MLSS) - Pinakamataas na sustainable na level ng lactate
  • Functional Threshold Pace (FTP) - Katumbas ng FTP sa cycling para sa pagtakbo

Bakit Mahalaga ang CRS

Ang critical running speed ang pundasyon ng advanced training load analysis:

  • Training Zones: Ginagawang personalized ang tindi ng pagsasanay base sa iyong katawan.
  • rTSS Calculation: Nagbibigay ng tumpak na Training Stress Score.
  • CTL/ATL/TSB: Kailangan para sa pagsubaybay sa fitness at fatigue.
  • Pag-unlad: Layuning sukat ng paglakas ng iyong aerobic fitness.

CRS Testing Protocol

üìã Standard na Protocol

  1. Warm-up

    15-20 minuto ng madaling takbo at dynamic drills.

  2. 1200m Time Trial

    Takbo nang todo (3 laps sa 400m track). Itala ang iyong oras.

  3. Ganap na Pahinga

    15-30 minuto ng mahinang jogging o lakad. Kailangan itong gawin para luminis ang lactate sa dugo.

  4. 3600m Time Trial

    Takbo nang todo (9 laps sa 400m track). Panatilihin ang parehong bilis hangga't maaari.
