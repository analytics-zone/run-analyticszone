Personvernerklæring

Sist oppdatert: 27. oktober 2025

Ditt personvern er beskyttet

Run Analytics samler ikke inn, lagrer eller overfører noen av dine personopplysninger. All databehandling skjer lokalt på enheten din.

Oversikt

Run Analytics er designet med ditt personvern som en toppprioritet. Denne personvernerklæringen forklarer vår forpliktelse til å beskytte dine personopplysninger og data.

Kort sagt: Vi samler ikke inn dataene dine fordi vi ikke trenger dem. Alt du gjør i appen forblir på enheten din.

Informasjon vi IKKE samler inn

Run Analytics samler ikke inn, overfører eller lagrer:

  • Personopplysninger (navn, e-post, telefonnummer, osv.)
  • Løpeaktivitetsdata (distanse, tempo, ruter, tider, osv.)
  • Helsedata (puls, kalorier, osv.)
  • Posisjonsdata eller GPS-koordinater
  • Enhetsinformasjon eller identifikatorer
  • Bruksstatistikk eller analyser
  • Krasjrapporter eller diagnostikk
  • Annen personlig eller sensitiv informasjon

Hvordan appen fungerer

Alle beregninger og databehandling i Run Analytics skjer utelukkende på enheten din:

  • Dine løpedata lagres kun i enhetens lokale lagring
  • Alle beregninger (tempo, splitter, soner, osv.) utføres lokalt
  • Ingen data sendes noensinne til våre servere eller tredjeparts tjenester
  • Ingen internettforbindelse er nødvendig for at appen skal fungere

Datalagring

Alle data du legger inn eller genererer i appen lagres utelukkende på enheten din ved hjelp av iOS sine lokale lagringsmekanismer. Disse dataene forblir under din fullstendige kontroll:

  • Dataene dine forblir på enheten din til enhver tid
  • Vi kan ikke få tilgang til dataene dine
  • Hvis du sletter appen, fjernes alle data permanent fra enheten din
  • Data synkroniseres ikke til iCloud eller noen skytjeneste med mindre du eksplisitt aktiverer iOS app-sikkerhetskopier

Tredjeparts tjenester

Run Analytics bruker ingen tredjeparts tjenester, inkludert:

  • Ingen analysetjenester (Google Analytics, Firebase, osv.)
  • Ingen annonseringsnettverk
  • Ingen krasjrapporteringstjenester
  • Ingen sosiale medier-integrasjoner
  • Ingen betalingsprosessorer (appen er gratis)

Tillatelser

Appen kan be om følgende iOS-tillatelser kun hvis du velger å bruke spesifikke funksjoner:

  • HealthKit/Helse-app: Kun hvis du velger å importere treningsdata. Alle data forblir på enheten din.
  • Varsler: Kun hvis du aktiverer treningspåminnelser. Ingen data sendes utenfor enheten din.

Alle tillatelser er valgfrie og du kan tilbakekalle dem når som helst gjennom iOS-innstillinger.

Barns personvern

Fordi vi ikke samler inn noen data, er Run Analytics trygt for brukere i alle aldre, inkludert barn under 13 år. Vi anbefaler imidlertid veiledning fra foreldre for alle treningsaktiviteter.

Endringer i denne personvernerklæringen

Vi kan oppdatere denne personvernerklæringen fra tid til annen for å gjenspeile endringer i appen eller juridiske krav. Eventuelle endringer vil bli publisert på denne siden med en oppdatert «Sist oppdatert»-dato.

Viktig: Vi vil aldri endre vår kjerneforpliktelse til ikke å samle inn dataene dine.

Kontakt oss

Hvis du har spørsmål om denne personvernerklæringen eller appen, vennligst kontakt oss gjennom vår kontaktside.

Sammendrag

Run Analytics er en personvern-først app. Vi mener at dataene dine tilhører deg og bør forbli hos deg.

  • ❌ Ingen datainnsamling
  • ❌ Ingen sporing eller analyser
  • ❌ Ingen tredjeparts tjenester
  • ✅ 100% lokal behandling
  • ✅ Fullstendig kontroll over dataene dine
